# 文章素材库

**用途**：沉淀讨论中的可写文章主题；供品牌叙事、治理说明、战略表述、深度内容时调用。

---

## 一、可写成文章的主题

| 主题 | 来源 | 角度 |
|------|------|------|
| **平台身份** | 讨论沉淀 §8 | 聚火、贡献者非攫取者、不拿大利润头、血液循环 |
| **分配机制** | 讨论沉淀 §9 | 无人持股、逆向投票、匹配机制、科学分配、公平可核查 |
| **自给自足** | 讨论沉淀 | 血液循环、平台强则生态强、可持续、公益方向 |
| **艺术供需桥** | 讨论沉淀 §7 | 供需不畅、平台做桥、何时是 right time |
| **生产链聚火** | 讨论沉淀 §7 | 平面→3D→金饰、多节点协作、软件实现协调 |
| **GAIA 角色** | 讨论沉淀 §13 | 无经济实体、最高精神指导、过渡门户、协调者 |
| **信任与牺牲** | 讨论沉淀 §8.2 | 可暂牺牲经济利益，不可牺牲信任；协调的边界 |
| **劫持防范** | 讨论沉淀 §9.9 | 系统不被任何人劫持；分散节点、归属实体 |
| **个人意见最小化** | 讨论沉淀 §9.10 | 规则引导、每个人按角色走、Founder 意见占比最小 |

---

## 二、文章形式建议

| 形式 | 适用主题 | 受众 |
|------|----------|------|
| 品牌叙事 | 平台身份、聚火、贡献者 | 对外 |
| 治理/机制说明 | 分配机制、逆向投票、成功标准 | 对内/伙伴 |
| 战略/愿景 | 艺术供需桥、生产链协调、自给自足 | 投资人、合作方 |
| 哲学/价值观 | 信任边界、牺牲精神、个人意见最小化 | 深度内容 |
