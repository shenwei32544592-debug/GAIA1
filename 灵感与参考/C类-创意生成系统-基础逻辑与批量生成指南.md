# 创意生成系统：基础逻辑与批量生成指南

**用途**：创意灵感的早期市场；可勾选/屏蔽的 Prompt 组合 → 随机组合 → AI 跑图 → AI 自筛选 → 产出可用创意。  
**前提**：笔画不分家；书法、绘画、木工、电脑设计、logo、符号可统一于一套逻辑。

---

## 一、基础逻辑结构

```
变量（可勾选/屏蔽）→ 随机组合 → 创意画像/简报 → AI 生成效果图 → 批量跑图 → AI 自筛选
```

---

## 二、变量设计（可自由勾选或屏蔽）

| 变量类型 | 选项示例 | 说明 |
|----------|----------|------|
| **名字** | James / John / [自定义] | 名字意涵、象征元素随之变化 |
| **名字意涵** | 延续继承 / 恩赐官方 / … | 见名字语料库 |
| **风格** | 商务 / 复古 / 简约 | 影响元素组合 |
| **宣言类型** | 主权宣告 / 生活哲学 / 祝福 / 极简 | 见英文主权文本库 |
| **象征元素-物件** | 皇冠、羽毛笔、书本、盾牌、十字架、圣杯… | 按名字/风格筛选 |
| **象征元素-花卉** | 橡树叶、月桂、橄榄枝、百合、白玫瑰… | 同上 |
| **象征元素-幻想** | 星标、守护环、怀表、新月、星光… | 同上 |
| **象征元素-动物** | 雄狮、雄鹰、骏马、白鸽、羔羊… | 同上 |
| **载体/品类** | 书法 / 绘画 / logo / 符号 / 木工 | 表现形态 |

**生成页逻辑**：用户可自由勾选或屏蔽上述变量；系统按勾选范围随机组合。

---

## 三、随机组合 → 创意画像

| 步骤 | 动作 |
|------|------|
| 1 | 用户勾选/屏蔽变量范围 |
| 2 | 系统随机组合：名字 + 风格 + 宣言 + 象征元素 + 载体 |
| 3 | 输出「创意画像」：一段可读的创作简报 |
| 4 | 创意画像作为 Prompt 输入 → AI 生成效果图 |

**创意画像示例**：
> James + 复古 + Own Your Path + 皇冠、羽毛笔、星标 + 书法
> → 「James，复古风格，主权宣言 Own Your Path，结合皇冠与羽毛笔元素，星标点缀，书法表现。」

---

## 四、批量跑图与 AI 自筛选

| 步骤 | 动作 |
|------|------|
| 1 | 固定变量范围，随机组合生成大量「创意画像」（如 10,000 条） |
| 2 | 每条画像 → AI 生成效果图 |
| 3 | 电脑批量跑图（如 10,000 张） |
| 4 | AI 自筛选：从生成图中选出「有可能接收、可使用」的 |
| 5 | 产出：可用创意池，供艺术家/平台选用 |

**目标**：创意灵感的早期市场；通过量化、随机、筛选，获得可用的创意储备。

---

## 五、Prompt 模板（可替换变量）

```
[名字] + [风格] + [宣言文本] + [象征元素 1-3 个] + [载体]
风格要求：[留白/极简/饱满]；主权感；可签名。
```

变量替换后 → 生成具体 Prompt → 输入图像生成模型。

---

## 六、语料库依赖

| 语料 | 位置 |
|------|------|
| 名字意涵与象征元素 | `名字语料库.json` |
| 英文主权文本库 | `A类-签名级文本.md`；Still Becoming、Own Your Path 等 |
| 风格-元素映射 | 商务/复古/简约 → 元素组合 |
| **输出格式** | **JSON**；示例见 `名字→AI绘图提示词-输出示例.json` |

---

## 七、实施顺序

1. 完善变量与语料 → 2. 搭建生成页（勾选/屏蔽 + 随机组合）→ 3. 输出创意画像 → 4. 对接图像生成 → 5. 批量跑图 → 6. AI 自筛选逻辑

## 八、工具用法（name_to_prompts.py）

| 操作 | 命令 |
|------|------|
| 输入名字，默认生成 10 条 | `python name_to_prompts.py James` |
| 指定条数 | `python name_to_prompts.py John 20` |
| 新名字（语料库无） | `python name_to_prompts.py Kolyan 15` → 用通用祝福+元素兜底 |

输出：同目录下 `output_名字_时间戳.json`，内含 `prompt_en` / `prompt_cn` / style / blessing / elements。
