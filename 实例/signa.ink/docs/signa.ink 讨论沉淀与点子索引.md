# signa.ink 讨论沉淀与点子索引

**品牌策略（2025-02-13 决策）**：对外统一使用 **WhoAmI.Art**；GAIA 仅保留为内部治理/宪法术语。宪法继续保留「本宪法为 GAIA 最高规范」；WhoAmI.Art 为本宪法首个商业实例。对外文档（商业计划、宣讲、艺术家邀请函、官网版等）不出现 GAIA。

**用途**：将讨论中的点子结构化沉淀，供产品/IT/系统落地时检索。  
当 Founder 启动新任务时，AI 可据此提示：「你曾提出过类似点子」并指向对应章节。

**维护**：每次重要讨论后更新；按主题分类，便于检索。

**约定**：以下点子已嵌入系统，后续讨论涉及业务、产品、艺术家、创意、落地时，AI 须检索并明确「何时可用」——在适当时机可调用、沿用。

## 七、战略方向待决：艺术供需桥（未来战略讨论时拿出）

**用途**：未来任何战略讨论时，AI 须检索本节，提示 Founder：「你曾提出过艺术供需桥方向」，并协助判断「现在是 right time 吗？」

### Founder 的观察（2025-02-10）

| 维度 | 要点 |
|------|------|
| 供给侧 | 世界各地散落大量画得好但孤岛化的艺术家；供应链（人、产能）存在，但无好平台对接；艺术本身价值不如想象高 |
| 需求侧 | 普通人无大众平台直接「遇见」作品；艺术形式多，个人逛、选成本高，转化率低 |
| 结论 | 供需不畅非无供给，乃对接不畅；signa 能否做桥？专门、活跃的艺术交易/发现平台，让消费者按偏好对接全球艺术家 |

### 与现有战略的张力

| 你描述的方向 | 现有战略 |
|--------------|----------|
| 泛艺术发现 + 撮合交易（类 Etsy） | 签名与印记窄品类：贺卡、签名簿、练字卡、兰花题名 |
| 平台效应、撮合效率 | 创意载体规范、源头不可抹除、正义优先于效率 |
| 全球艺术家、多品类 | 签约源头艺术家、协议约束 |

### 何时拿出讨论

| 触发场景 | AI 动作 |
|----------|----------|
| Founder 说「战略讨论」「复盘方向」「考虑扩张」 | 检索 §七，提示 prior 思考，问「艺术供需桥方向，现在是 right time 吗？」 |
| Phase 2/3 规划、品类扩张、平台化决策 | 同上；并对照创意载体规范，评估哪些品类可延展、哪些受约束 |

### 生产链协调：平台聚火（2025-02-10 增补）

| 维度 | 要点 |
|------|------|
| 链条 | 平面→3D→金饰/珠宝；多节点（flat brush、文字转立体、3D 转换、高端加工）；传统市场难自发形成，地理分散（如法国、英国） |
| 平台角色 | 聚火、协调协作、透明分润、营销出口；每次整合可组合无数万花筒 |
| 价值 | 软件才能实现；单靠线下无法达成 |

## 八、平台身份与利润观（宪法级原则，2025-02-10）

**用途**：战略、产品、分润设计时须检索，确保平台身份不偏离。

| 原则 | Founder 原话要点 |
|------|------------------|
| 平台唯一核心 | 聚火/整合——平台需要做的只有这一件事 |
| 不拿大利润头 | 没必要；可组合无数万花筒，机制在此 |
| 信任 | 众人把信任交给平台；平台因凝聚力而生 |
| 品牌价值 | 每次整合即增值，即便不抽成 |
| 方向 | 向人类公益发展；资金足够后并不需要再攫取 |
| 不做炫耀式公益 | 很多人做公益为炫耀——不适合我们 |
| **宪法逻辑** | **做贡献者，不做攫取者** |
| 血液循环 | 必须覆盖能源、维持基本运转；理想在知识池中；平台强则生态强 |

### 8.2 协调与牺牲的边界

| 可暂时牺牲 | 不可牺牲 |
|------------|----------|
| 经济利益 | 信任 |
| 个人便利、短期回报 | 新船/事业存续、参与者信心 |
| 牺牲可量化、有回报预期 | 信任不可透支；信任一失，随时崩塌 |

## 九、无人持股 + 按贡献与投资分润（治理与法律设计，2025-02-10）

**用途**：经济实体注册、分润设计、治理结构时检索；可逐步落地的路径。**当下**：注册经济实体。

### 9.1 核心设定

| 原则 | 内容 |
|------|------|
| 无人持股 | 经济实体内部无人持有股权/股份 |
| 按贡献分润 | 按贡献程度分配收益 |
| 按投资分润 | 投资层级可获得相应分润 |
| 联盟范围 | 贡献与分润可跨越单一平台，在联盟层面核算 |

### 9.2 分润权来源（合同约定，非股权）

| 来源 | 分润依据 |
|------|----------|
| 贡献 | 创意、技术、劳动、运营 → 贡献评估 / 贡献凭证 |
| 投资 | 资金、资源 → 投资协议、回报条款 |
| 参与 | 艺术家、合作方 → 分成协议 |

- 实体结构：LLC / 非营利 / 合同网络，按法域选择

### 9.10.1 决策前提醒（2025-02-10）
