

# 🎬 signa.ink「LOGO 定锚片」
## One-page Shotlist + Sound Design
**Film Type:** Genesis Anchor Film  
**Duration:** 22–25s  
**Tone:** Cold / Precise / Monumental  
**Aspect:** 16:9（主版）｜可安全裁切 9:16  
**Color:** Black / Brass / Paper White

---

## 🎥 SHOTLIST（按时间顺序）
---

### **SHOT 01｜ORDER APPEARS（秩序显现）**
**Time:** 0:00–0:04

+ **画面**
    - 纯黑背景
    - 极细几何线条（LOGO 的局部结构）逐条对齐、锁定
    - 运动极慢，像机械校准
+ **摄影**
    - 数字动画或实拍+后期
    - 无景深变化，绝对冷静
+ **字幕（可选）**
    - _When copying is free, verification becomes scarce._
+ **音效**
    - 微弱金属机械对位声（低频）
    - 无音乐

---

### **SHOT 02｜FLOW ENTERS ORDER（流动进入秩序）**
**Time:** 0:04–0:10

+ **画面**
    - 墨线 / 笔触进入几何网格
    - 线条延展 → 碰触边界 → 被约束 → 收笔
    - 中途 **0.3s 插入约束检测界面红标闪现**
+ **摄影**
    - 微距（Macro）
    - 慢动作（60–120fps）
+ **关键要求**
    - 不要“美化”，要“克制”
    - 让观众感到：**不是自由，是守法**
+ **音效**
    - 笔触摩擦声（干净、无混响）
    - 极轻“警示音”一闪（非UI感，像仪器反馈）

---

### **SHOT 03｜PHYSICAL PROOF I（纸）**
**Time:** 0:10–0:13

+ **画面**
    - 钢笔笔尖压入高棉浆纸
    - 纸纤维下陷、回弹
+ **摄影**
    - 超微距
    - 浅景深，焦点锁在接触点
+ **灯光**
    - 侧逆光，强调纸张纹理
+ **音效**
    - 清晰纸纤维摩擦声（ASMR级）

---

### **SHOT 04｜PHYSICAL PROOF II（金属）**
**Time:** 0:13–0:16

+ **画面**
    - 黄铜表面刻蚀 / 压印
    - 光线扫过凹陷边缘
+ **摄影**
    - 微距
    - 轻微横向移动（Slider）
+ **灯光**
    - 冷光 + 窄条反射
+ **音效**
    - 极轻金属切削 / 压印声
    - 不要火花音效（避免工业片感）

---

### **SHOT 05｜PHYSICAL PROOF III（封存）**
**Time:** 0:16–0:18

+ **画面**
    - 火漆滴落 → 印章压下 → 凝固
+ **摄影**
    - 慢动作
    - 构图居中，仪式感
+ **灯光**
    - 暖色，但整体仍偏冷
+ **音效**
    - 火漆轻微气泡声
    - 印章落下的“闷响”

---

### **SHOT 06｜ANCHOR（定锚）**
**Time:** 0:18–0:22

+ **画面**
    - 档案盒 / 金属 Key / 认证物件
    - LOGO 完整出现（不动画、不炫技）
    - 小字浮现（可选）：
        * `Protocol v1.1 | Freeze ID: EXP-001`
+ **摄影**
    - 定机位
    - 无多余运动
+ **音效**
    - 档案盒合上的低频声
    - 所有环境音瞬间静止

---

### **SHOT 07｜SIGNATURE END（留白）**
**Time:** 0:22–0:25

+ **画面**
    - 纯黑
    - LOGO + 文案
        * **signa.ink**
        * _Defining Your Mark._
+ **摄影**
    - 静帧
+ **音效**
    - 无音乐
    - 仅留一声极轻“空气回落”

---

## 🎧 SOUND DESIGN 总原则
+ **拒绝背景音乐旋律**
+ 只使用：
    - 机械低频
    - 材质真实声
    - 空气 / 静默
+ 声音要让人觉得：

“这是一个被记录的事实，不是被包装的广告”

---

## 🎞️ 剪辑与节奏要求
+ 全片 **≤ 25 秒**
+ 无快切
+ 无视觉炫技
+ 每一帧都要像“证据”

---

## ✅ 导演交付标准（Checklist）
+ LOGO **不是装饰，是秩序来源**
+ 看完不像广告，更像一次**宣誓**
