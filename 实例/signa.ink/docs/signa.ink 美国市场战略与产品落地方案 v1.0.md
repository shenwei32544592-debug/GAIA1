## 一、方案来源与定位

本方案由 Founder 与 AI Steward 多轮对话沉淀而成，整合：

- **软性思维**：平台设计中的柔性、可演进逻辑
- **产品思维**：具体 SKU、定价、履约
- **玩转思维**：游戏、漏斗、传播机制

**目标**：从「讨论」回到「起点」，形成可执行、可验证的系统与产品方案。

## 二、战略总览

### 2.1 核心判断

| 维度 | 结论 |
|------|------|
| 市场 | 美国个性化礼品、情感表达有真实需求 |
| 供给 | 中国艺术家成本可控，可支撑跨境交付 |
| 切入点 | 签名 + 祝福 + 礼物形态 |
| 原则 | 先单品、先盈利、再扩品 |

### 2.2 战略路径

```
Phase 1：单品验证（$9.9 超级单品）
    ↓
Phase 2：渠道放量（白板 + 游戏 + 漏斗）
    ↓
Phase 3：档位扩展（$19.9 / $29.9）
    ↓
Phase 4：平台化（创业广场 + 投资结构）
```

## 三、产品体系

### 3.1 超级单品（Phase 1 主推）

| 元素 | 设计 |
|------|------|
| **形态** | 固定 8 寸礼盒 + 贺卡 |
| **内容** | 国画/花卉 + 祝福语 + 名字，全预写 |
| **材质** | 皮革或古风纸，艺术家设计 |
| **名字区** | 特别设计，可烫金或素雅 |
| **价值感** | 精致礼盒 + 可收藏/可展示 + signa 露出 |
| **定价** | $9.9 落地 |

**逻辑**：盒内一切可见 = signa 品牌在场；$9.9 锚点，支撑转化。

### 3.2 半定制贺卡（$9.9–14.9）

| 层级 | 内容 | 执行方 |
|------|------|--------|
| 上半 | 手写名字 + 祝福语 + 小花 | 艺术家 |
| 下半 | 留白 | 送礼人自写 |
| 结构 | 背面立架，可站立 | — |

**定位**：名字 + 祝福结合，大部分人偏好；参与感强，不显廉价。

### 3.3 档位结构

| 档位 | 内容 | 价格 | 定位 |
|------|------|------|------|
| 入口 | 半定制贺卡 / 超级单品礼盒 | $9.9 | 破冰、走量 |
| 进阶 | 裱框、书法+印章、机器人练字卡 | $19.9 | Flow 入门 |
| 高端 | 国画小品 + 题名、艺术家亲签 | $29.9+ | 收藏、深度定制 |

### 3.4 批量签名簿（婚礼/聚会）

| 元素 | 说明 |
|------|------|
| 形态 | 国风签名簿/功德簿 |
| 场景 | 婚礼、小聚会、活动 |
| 订单 | 单次 30–50 张，顾客自选朋友名字 |
| 成本 | 单张约 $1，顾客付快递 |
| 利润 | 单笔约 $5，价格-价值感好 |

**策略**：主攻此市场；个性化定制作为升级路径。

### 3.5 辅助产品线

| 产品 | 作用 | 与 signa 关系 |
|------|------|---------------|
| 磁信白板 | B2B 礼品 + 广告渠道 | 独立业务，补贴 signa 获客 |
| 兰花/花卉小品 | 画家成本约 $4，可题名 | $29.9 档位组成部分 |

## 四、渠道与获客

### 4.1 渠道矩阵

| 渠道 | 形态 | 目标 |
|------|------|------|
| 磁信白板 | 说明书背面 signa 广告 | 触达家庭，长留存 |
| 在线拼图游戏 | H5/小程序，分级难度 | 攀比分享，导流 signa |
| 定向广告 | 华人社区、礼品场景 | 可控获客 |
| 诊所/中介合作 | B2B 推荐 | 感谢礼场景 |

### 4.2 漏斗设计

```
免费模板选择 → 「选好后设计师可书写」
    ↓
网站送 $20 优惠券（名义免费）
    ↓
$9.9 实体：签名卡 + 礼盒 + 创作视频
    ↓
$19.9 裱框版 / $29.9 小品画
```

### 4.3 游戏获客

| 设计 | 说明 |
|------|------|
| 形态 | 华容道式拼图 + 透明双板显像 |
| 平台 | 线上 H5，易分享 |
| 机制 | 分级难度 → 「极少数人达到」→ 分享挑战 |
| 转化 | 过关 → signa 权益 / 免费设计 |
| 投入 | < 5 万 RMB |
| 复用 | 可换素材、换品牌，多市场复用 |

## 五、履约与成本

### 5.1 履约模式

| 模式 | 适用 | 操作 |
|------|------|------|
| 限量款/标准款 | 可预测品类 | 美国仓备货 → 贴单即发 |
| 定制款 | 有量时 | 集单 → 国际快递 3–5 日到美国仓 |
| 单件直发 | 量极少 | 不主推，成本难摊 |

### 5.2 成本逻辑

| 条件 | 结论 |
|------|------|
| 量大 | 国际运费可谈判，单均约 $5–7 毛利 |
| 量小 | 单均成本高，需先打通闭环 |
| 原则 | 体量发货，集单降本 |

### 5.3 支付与交付

| 环节 | 设计 |
|------|------|
| 支付 | Stripe + PayPal，必选 |
| 运费 | 美国境内包邮或透明含运费 |
| 时效 | 明确告知，管理预期 |

## 六、系统设计：创业广场与投资结构

### 6.1 创业广场（Entrepreneurship Square）

| 原则 | 说明 |
|------|------|
| 集体创新 | 不依赖单一创意来源 |
| 艺术家联盟 | 多人提想法、多人执行 |
| 公式化 | 创业流程标准化、可复制 |
| 联合生产 | 广场内生产者协作 |

### 6.2 投资结构

| 要素 | 设计 |
|------|------|
| 项目制 | 按具体项目投资，非股权 |
| 认投 | 可投部分或整单；簿记、设计师、任何人可认投 |
| 认投者 | 认投即成为投资者；认投可撤回（项目启动前） |
| 分配 | 利润 → 投资者 + 执行者 + 贡献者 |
| 平台 | 协调、分配、标记，不截留 |
| 合规 | 资金、账户、协议留痕 |
| 性质 | 投资协议，非股票 |

### 6.3 利益链与变现路径（按节点类型）

| 节点类型 | 变现方式 | 平台角色 |
|----------|----------|----------|
| 艺术家 | 平台内服务合作 → 按协议分成 | 雇主/合作方 |
| 消费者 | 积分兑换商品 | 商家 |
| 认投多者 | 转赠积分给亲友 | 提供机制 |
| 用户间 P2P | 私下议价、私下付款；平台仅提供积分过户 | 不参与资金流 |

**合规逻辑**：P2P 资金在平台外完成，平台不触及，规避支付/证券相关责任。

### 6.4 利益链保留与合规策略

| 策略 | 说明 |
|------|------|
| 内部账本先行 | 全球统一规则，不直接声称货币/投资 |
| 表层：游戏/积分 | 以平台内行为呈现，降低法律摩擦 |
| 底层：接口保留 | 在合规法域接通真实结算 |
| 法律形态 | 预购/众筹/会员积分/合作分成，避免证券定性 |
| 类比 | 航空里程、Kickstarter、健身房储值 |

### 6.5 与 GAIA 宪法对应

| 宪法原则 | 系统体现 |
|----------|----------|
| 共识高于权力 | 创业广场集体参与 |
| 权利与义务对等 | 投资、贡献、分配挂钩 |
| 源头不可抹除 | 创意、贡献、分配可追溯 |

## 七、执行路径

### 7.1 Phase 1：单品验证（0–3 个月）

| 步骤 | 动作 |
|------|------|
| 1 | 确定超级单品：8 寸盒 + 贺卡，物料统一 |
| 2 | 打通一条主渠道：白板说明书 / 游戏 / 小规模投放 |
| 3 | 落地页 + Stripe/PayPal + $9.9 首单 |
| 4 | 50–100 单验证：转化率、成本、履约 |

### 7.2 Phase 2：闭环跑通（3–6 个月）

| 步骤 | 动作 |
|------|------|
| 5 | 稳定艺术家供给：2–3 人长期合作，月入 2–3 万 RMB |
| 6 | 集单履约，控制国际物流成本 |
| 7 | 单均毛利 $5–7，利润分配清晰 |
| 8 | 艺术家 / 平台 / 物流 / 渠道分配固化 |

### 7.3 Phase 3：放量（6–12 个月）

| 步骤 | 动作 |
|------|------|
| 9 | 增加渠道：更多白板合作、游戏推广、定向广告 |
| 10 | 美国前置仓（量稳定时） |
| 11 | 扩展到更多区域 |

### 7.4 Phase 4：平台化（盈利后）

| 步骤 | 动作 |
|------|------|
| 12 | 系统稳定、盈利后评估新产品线 |
| 13 | 创业广场、投资结构逐步落地 |
| 14 | 新品沿用同一逻辑：单品、通用物料、验证后推广 |

## 八、关键约束与红线

### 8.1 产品红线

| 红线 | 说明 |
|------|------|
| 机器人复写 | 须标明「练字模版」，非艺术家亲签 |
| 批量视频 | 可复用，但文案不宣称「专为你一人」 |
| 价格展示 | 不用虚假高价制造锚定，避免误导 |

### 8.2 财务约束

| 约束 | 说明 |
|------|------|
| 白板 | 至少保本，补贴有上限 |
| 验证期 | 6–12 个月，转化不达则调整 |
| 现金流 | 找到接受账期的合作方 |

### 8.3 艺术家承诺

| 目标 | 数值 |
|------|------|
| 长期合作者 | 月入 2–3 万 RMB |
| 原则 | 订单稳定 → 收入稳定 → 人才能留 |
