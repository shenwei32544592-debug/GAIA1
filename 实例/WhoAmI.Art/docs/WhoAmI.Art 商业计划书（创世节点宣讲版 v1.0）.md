# 开篇：一句话定位

WhoAmI.Art 是一个以 **数字身份授权（DIL）** 为核心产品、以 **参数化分配引擎** 为经济基础、以 **分层自治治理** 为秩序内核的全球艺术商业联邦。

**我们不是在搭一个电商平台，而是在铸一套制度机器**——让艺术家的信用、技艺与全球需求对接，让分配可追溯、权力不腐化、系统可持续。

**本文档面向**：每一位可能的合作者。无论你是艺术家、技术伙伴、运营节点还是投资人，读完此文，你将理解我们的市场判断、逻辑推演、制度设计与未来路径。

# 第一章：市场分析

## 1.1 供需为何失衡

| 供给侧 | 需求侧 | 缺口 |
|--------|--------|------|
| 全球散落大量优质艺术家、非遗传承人、手艺人 | 消费者追求个性表达、身份印记、定制化 | **缺乏有效撮合** |
| 有技艺、有作品、有成熟供应链 | 愿为高性价比的个性服务付费 | 供需距离太远 |
| 中国抖音直播已成功的艺术家，争着想出海 | 海外市场缺乏对接通道 | 流量在手，缺转化与履约 |

**核心痛点**：有技艺的人无法与全球消费者直接对接；传统路径要么被中间商压价，要么自建成本过高。

## 1.2 谁在等我们（市场现实）

- **大量在中国抖音做直播、通过非遗艺术亲自操演已成功的艺术家**，主动寻求与平台合作。
- 他们已有：**成功的供应链、现成素材、自有流量**。
- 只需把**原始素材**交给美国运营节点 → 流量自然产生 → 导入接单平台 → 艺术家直面客户的订制或成品选购。
- **流量不是我们求来的，是艺术家带来的**；我们负责转化、履约与分配。

## 1.3 市场机会

| 维度 | 说明 |
|------|------|
| 柔性生产趋势 | 消费者不再满足标准化，追求个人印记 |
| 信息透明时代 | 垄断难以为继，公平分配成为可能 |
| 出海刚需 | 中国成功艺术家急需合规、可持续的全球化通道 |
| 制度空白 | 缺乏「可追溯分配 + 艺术家主权」的协作基础设施 |

# 第二章：逻辑分析

## 2.1 完整逻辑链

```
供需失衡（艺术家触达困难）
    ↓
平台撮合 + 履约 + 分配 + 透明
    ↓
吸引优质艺术家（他们带来流量与供给）
    ↓
流量导入接单平台 → 艺术家直面客户
    ↓
互哺飞轮：更多优质供给 → 更多消费者 → 平台壮大 → 更多人受益
```

## 2.2 价值主张

| 对象 | 我们提供 |
|------|----------|
| **艺术家** | 出海渠道、标准化履约、可追溯分配、主版权保留、Kill Switch |
| **消费者** | 高性价比的个性身份印记、全球艺术家选择、可信服务 |
| **技术/运营节点** | 贡献确权、衍生业务优先权、制度化权益 |

## 2.3 为何能成

- **供给端已就绪**：抖音成功的非遗艺术家争着想合作；供应链与素材在手。
- **流量不愁**：艺术家自带流量；我们做转化与履约，不做从零获客。
- **制度差异化**：可追溯分配、艺术家主权、无人攫取过度——这是信任的基础。

# 第三章：制度解析

## 3.1 分配引擎（三张表）

### 订单经济表

| 池子 | 比例 | 说明 |
|------|------|------|
| 艺术家利润部分 | 65% | 直接归该单艺术家 |
| Anchor Pool（创世公民） | 5% | 创始节点系统级红利 |
| 公共运营池 | 20% | 运营、IT、扩张；取之于民，用之于民 |
| FCR（风险基金） | 10% | 退款、争议、FX 缓冲 |

### 贡献记账表

| 类型 | 内容 | 兑现 |
|------|------|------|
| A 类 | 订单、交付、履约 | 可参与分润 |
| B 类 | 讨论、建议、传播 | 只记账，延迟兑现；设上限 |

### 批次权益表

| 批次 | 权益 | 防修改 |
|------|------|--------|
| 创世公民 | 5% Anchor Pool、起草委员会席位、Kill Switch | 写入协议；变更可追溯 |
| 后续 | 无 Anchor 或递减 | 同上 |

**创世公民 机制**：Floor% Bucket、Cap（200%）、Catch-up；Time Decay 18 月→50%。

## 3.2 治理结构（三权分立）

| 权 | 主体 | 职责 |
|----|------|------|
| **执行权** | Founder | 产品、技术、商业节奏；72h 紧急暂停；**不能改分配规则** |
| **自治权** | Domain Council | 按门类设立；新人准入、竞争协调、行业伦理；不碰钱、不碰码 |
| **分配权** | 参数化引擎 | 硬编码、自动执行、不可篡改 |

**分阶段治理**：Phase 0 Founder 强治理 → Phase 1 引入监督 → Phase 2 委员会与投票。治理复杂化以订单跑通为前提。

## 3.3 准入与社会资本

- **提名制**：新人须由现任成员提名；提名人承担背书责任。
- **评分**：艺术成熟度 40%、授权意识 20%、交付可靠性 20%、生态多样性 20%；≥75 通过。
- **背书责任**：新人违约，提名人承担连带补偿。
- **创世公民 内部分层**：运营锚点（高频履约、配合度高、自带素材）vs 名誉锚点（艺术高度、准入背书）；验证期优先运营锚点。详见《创世公民与准入分层》。

## 3.4 去中心化仲裁（DAS）

- 分级仲裁（1 法官+3 陪审 至 3 法官+9 陪审）；败诉方承担成本。
- 判断正确→被选概率提升；不惩罚错误；全流程可回看。
- 形成平台判例法文化。

## 3.5 创世节点权益概览

| 角色 | 权益 |
|------|------|
| **Artist / Anchor** | 65% 直接分润、5% Anchor Pool、Kill Switch、收款自由 |
| **IT / 创世技术官** | 创世公民 身份、5% Anchor 分配权重、里程碑现金、贡献确权、衍生业务优先权、遗产分红、禅让后保留权益 |
| **运营 / 社会资本节点** | 创世公民 Social Capital Node、公共运营池专项贡献权重、入口筛选权、社会资本入股、结算先于分账、失败不连坐 |
| **投资人** | 投资款用于建设；回报从约定结构提取；与运营分润分离 |

## 3.6 投资、储备金与基础设施

- **投资准入**：可接受独立投资人；投资款用于建设，与分润分离。
- **系统储备金**：公共运营池内约 2–3%；用途优先（升级、合规、人才、激励）；分红后置（日≥200 单且盈利后）；可审计。
- **资本流与货流分离**：投资端与运营端独立设计；财务、物流以第三方为主。

# 第四章：发展路径

## 4.1 起步策略（一句话）

**签第一位 Anchor → 1 周内上线 → 跑通首单到账 → 让他安心 → 用结果接触下一位。**

## 4.2 阶段与阈值

**SST（System Stable Threshold）**：系统稳定阈值 = **日均 200 单**且连续稳定 30 天。  
**制度约定**：任何治理复杂化、分红释放、职位禅让，均不得早于 SST 达成。

| 阶段 | 日订单 | 定位 |
|------|--------|------|
| 验证期 | <50 | 首位跑通、模型验证 |
| 起步期 | 50–200 | 多艺术家、多品类试跑 |
| **正常运营（SST 达成）** | **≥200** | 稳定规模、可持续运转 |
| 扩张期 | >500 | 扩展品类与市场 |

## 4.3 30 天闪电闭环

1. **[1–7 日]** 锁定 Anchor；签署 MOU
2. **[8–15 日]** 建立数字工厂；3 个 SKU；交付 SOP
3. **[16–25 日]** 合规渠道；Stripe；版权合同
4. **[26–30 日]** 压力内测；跑通支付→分配→艺术家收款

## 4.4 产品路径

| Phase | 形态 | 说明 |
|-------|------|------|
| 1 | 纯数字/授权 | 零物流、可标准化、版权许可费 |
| 1.5 | 数字+限定实体 | 美国采购、国内签样、美国分装 |
| 2 | 更多实体 SKU | 定制书法、匾额、空间陈设 |

**0 元入口**：Identity Draft（横版模板）→ 升级 $99–$199 正式授权、$499+ 实体、$1000+ 定制。

# 第五章：未来规划

## 5.1 衍生业务

- 履约插件（美国仓储、智慧物流）
- 供应链系统（国内非遗、实体周边）
- 合规结算工具、企业定制方案
- 创始节点享有**优先承接权**；须遵守最惠国条款与交叉审计

## 5.2 禅让与长寿

- CEO/CTO **非终身制**；日≥200 单后可动议引入职业经理人。
- 执行权移交后，创始节点保留 创世公民 分润权与储备金分配动议权。
- 成功培养继任者，可获传承奖金。
- **制度高于个人**：代码会折旧，宪法会常青。

## 5.3 规模化

- 日 200 单为正常运营分水岭；此前聚焦验证与冷启动。
- 分配比例守恒；Anchor 有 Cap；公共运营池须覆盖扩张成本。
- 资本流与货流分离；非核心外包；核心内聚。

# 第六章：风险与兜底

## 6.0 风险清单

| 风险 | 兜底 |
|------|------|
| 供给质量 | 首批只收可标准化交付者；试单/SLA |
| 消费者不买 | 礼品场景：毕业、升职、婚礼 |
| 退款/纠纷 | 明确边界；FCR+仲裁 |
| 贡献值内斗 | A/B 类；B 类延迟；上限 |
| 跨境履约 | 先数字；实体 1 SKU 1 仓 |
| 大咖信任 | 名誉保护；Kill Switch |
| 合规 | 分润仅来自真实业务；版权许可费 |

## 6.1 Founder 自我约束宣言

作为本联邦的发起人，Founder 显式承诺受以下条约约束：

1. **比例不动**：不以个人意志更改已硬编码的 65/5/20/10 分配比例。
2. **资金隔离**：在未达成 SST 前，不从系统储备金中提取个人利润。
3. **信息透明**：所有 Founder 级紧急暂停权（72h）必须在事后 24h 内产生 Append-only 治理日志记录。
4. **违约仲裁**：若 Founder 违反上述原则，Domain Council 与创世节点有权共同启动审计程序。

# 第七章：如何加入

## 7.1 艺术家

- 若已在抖音等平台成功，且有出海意向：提交作品与供应链信息；经 Domain Council 审核后可进入。
- 首批 5–10 位为 Anchor Artist，享有 创世公民 权益。
- **验证期策略**：中坚力量（SOP-Ready、自带物料）优先于顶级大师；先跑通利润，再追求名声。详见《创世公民与准入分层》。

## 7.2 技术伙伴（IT）

**心理锚点（三句话）**：安全（分配规则写死在系统里，谁也赖不掉）、身价（创世公民 永久信用底座，入股规则非干活）、未来（禅让接管，退拿红利）。

- 参与 MVP 建设；按里程碑获取现金；贡献绑定 RuleID；享有衍生业务优先权与遗产分红。
- 详见《核心技术节点合作备忘录（摘要版）》（含三句话与商人视角拆解）、《IT 伙伴 10 分钟讲稿》《创世技术官里程碑与现金协议》《创始伙伴协作协议（核心摘要）》。

## 7.3 运营 / 社会资本节点

**定位**：第二核心节点 = 让制度第一次被现实世界承认的人；不是打工型运营，而是用真实人脉、圈层、信用背书带人进来。

**心理锚点（三句话）**：安全（你带人进来，我负责制度和结算兜住，你不替平台背信誉风险）、身价（社会信用换首批成功案例，永久记账）、未来（跑通后站在入口筛选，从消耗型→守门型→定价型）。

- 按 A 类贡献参与公共运营池分配；每个成功引入的 Anchor/GMV 可追溯记账。
- 详见《第二核心节点（运营/社会资本）合作备忘录》《运营节点 5 分钟讲稿》。

## 7.4 投资人

- 投资款用于建设；回报结构单独约定；与运营分润分离。

# 结语

WhoAmI.Art 正在构建的是一套可以承载判断力、信用与长期协作的制度机器。通过结构化设计，让正确的商业行为更便宜，让错误的投机行为更昂贵。

**核心验证指标**：第一笔跨境付款能否合法、安全地到达艺术家账户——这比任何 Landing Page 都重要。
