# WhoAmI.Art 内部流动性与合作节点治理补充

**用途**：启动期现金流、自愿延迟提现、营销/招商节点、一级伯乐、回购权、创始人风险对冲  
**版本**：v1.0  
**性质**：制度补充，可纳入宪章附件

---

## 一、内部流动性循环（启动期）

### 1.1 逻辑

核心成员既是**开发者**又是**债权人**。不要求掏现金，只承诺「应提现的款等一等」——钱已是客户付过的确定款项，心理账户为「存定投」而非「支出」。

### 1.2 账户状态

| 账户 | 说明 |
|------|------|
| 初始投入账户 | 实际垫资（物料、域名、服务器） |
| 应计分润账户 | 5%/20% 中属于你但被「冻结」的部分 |
| 可提现余额 | 现金流转正后解锁 |

### 1.3 水位线释放

| 水位 | 触发 | 动作 |
|------|------|------|
| A 生存线 | FCR < 1 万美元 | 分润 100% 冻结 |
| B 偿还线 | FCR 满额且现金流转正 | 分批解冻，优先偿还本金 |
| C 分红线 | 本金还毕 | 应计分润按月转入可提现 |

### 1.4 未提现的理财补贴

未提现余额 × (Mercury 月化利率 + 1% 创始节点奖励) = 当月补贴。用未来结余优先支取，名义为「利息支出」，抵减 LLC 应税所得。

## 二、自愿延迟提现与高收益认领

### 2.1 双路径

| 路径 | 对象 | 规则 |
|------|------|------|
| **标准提现** | 普通艺术家、供应商 | 14 天后随时取走 |
| **红利认领** | 核心创始人、创世公民 | 点击「认领延迟」，进入影子锁定期 |

### 2.2 高收益回报率

设 12%–18% 年化，以日复利记录。本质：用未来结余优先支付，不掏现金。

### 2.3 创始人兜底

若无人认领，额度静默滑入 Founder 与 IT 伙伴的「影子债权池」——名义为「流动性担保责任」，实际捕获最高收益。**名利双收**：名 = 自愿殿后；利 = 无人领则由创始人收。

### 2.4 认领限额与防挤兑

设总认领额度上限；超额需排队或降档。确保未来结余能覆盖利息。

## 三、营销节点（Growth Node）

### 3.1 定位

邀请网红、自媒体成为**营销节点**，非外包。用**分润**替代**买流量**。

### 3.2 分润逻辑

| 订单类型 | 公共运营池 20% 拆分 |
|----------|---------------------|
| 非营销引导 | 20% 全归联盟运营 |
| 营销引导（Affiliate Link） | 10% 联盟运维 + 10% 营销节点 CPS |

### 3.3 同样适用延迟认领

营销节点可选择立即提现或认领高收益，将佣金留在系统内滚动。

### 3.4 管理

专属 RefID、透明看板、CTR/CVR 红线、优胜劣汰。

## 四、招商节点与一级伯乐

### 4.1 全员招商

**招商功能**内化为系统通用模块，任何节点（含艺术家）均可通过专属链接邀请新人。艺术家最懂艺术标准与成本，天然质量过滤器。

### 4.2 一级限制

只允许**一级**提成。A 招 B，B 产生销售则 A 拿伯乐奖；B 招 C，C 的销售与 A 无关。防佣金塌陷、防传销化。

### 4.3 伯乐奖金

从 20% 公共运营池划 2%–3%；仅在产生成交时支付。成功费（Success-based Carry）。

### 4.4 审核权归属

上架权交给**分平台艺术家委员会**审核；开新门类须达到一定等级的艺术家联名动议，并经委员会 2/3 投票，讨论可行性、风险、冲突。

## 五、委员会与荣誉治理

### 5.1 贡献分（RP）

| 行为 | RP |
|------|-----|
| 审核新项目 | +10 |
| 参与新门类投票 | +5 |
| 成功保荐新人（过 1x SST） | +50 |

### 5.2 名誉对价（非现金）

不直接分钱，给予**名誉或贡献分**，用于：
- 出席宴会、代表联盟的外交权
- 优先清算权（极端波动时）
- 特权准入（联名、策展等）

RP 达标的自动优先，比现金更具社会资本杠杆。

### 5.3 名誉分半衰期

半年无审核或保荐人表现低迷，RP 自动衰减。防平庸化与派系斗争。

## 六、回购权（创世公民 份额）

### 6.1 适用场景

| 场景 | 说明 |
|------|------|
| **资产污染** | IT 伙伴遭遇债务、离婚，债权人可能执行其 2.5% 份额 |
| **和平退出** | 财富自由后欲套现开启新人生 |
| **税务/身份** | 移居、入籍导致退出税或合规切割 |
| **愿景漂移** | 双方方向无法兼容 |

### 6.2 回购公式

```
Buyback_Price = LTM_Payout × Multiple × (1 - Discount_Rate)
```

Multiple 通常 3–5 倍；被迫退出 Discount 低，主动提前退出 Discount 高。

### 6.3 本质

回购权是**保险**，不必动用，但一旦外部（债权人、法院）敲门，可优先回购保住主权。

## 七、创始人自身风险对冲

### 7.1 创始人也可能带来的风险

| 风险 | 对平台冲击 |
|------|------------|
| 身份/签证波动 | 账户冻结、主权中断 |
| 专业身份洁癖 | CPA 等受影响时可能被迫切割 |
| 家庭/继承 | 治理权碎片化、多头领导 |

### 7.2 自我对冲

| 机制 | 说明 |
|------|------|
| **影子受托人** | 2.5% 由家庭信托或数字遗嘱合约持有；出事后治理权平移 |
| **公益基金回购** | 若创始人必须切割，份额由公益基金回购，主权回流系统 |
| **Dead Man's Switch** | 若 CEO 连续 X 天未登录审计后台，治理权自动移交预设对象 |

## 八、执行节点（壳公司）授权书要点

- 公司为「代收代付代理人」，无资产所有权、无分配决策权、无规则解释权
- 禁止修改 65/5/20/10
- 禁止将域名、源码、RuleID 列为公司资产
- 禁止截留应分配给艺术家或积累池的资金
- 联盟代表会有权随时撤回授权；24 小时内移交账户管理权
