# WhoAmI.Art 完整商业计划书（联邦治理版 v1.1）

**版本**：1.1  
**日期**：2025-02-12  
**用途**：投资人 / 合伙人 / 核心协作方；整合多轮深度对话中的可验证路径、三张表、风险兜底与分阶段治理。

**v1.1 优化要点**：压缩宏大叙事比例；将「首批 30 天可验证路径」提升至核心篇幅；补齐订单经济表、贡献记账表、批次权益表；增加风险清单与兜底；明确分阶段治理；补充 0 元入口与升级路径、冷启动策略。

## 一、一句话定位（投资人可 30 秒理解）

WhoAmI.Art 是一个以 **数字身份授权（DIL）** 为核心产品、以 **参数化分配引擎** 为经济基础、以 **分层自治治理** 为秩序内核的全球艺术商业联邦。

它通过制度设计，让正确的协作成本更低，让个体信用成为可流转资产；**首批 30 天的验证目标** 是跑通 1 条业务闭环：1 个爆款场景 + 1 种交付形态 + 1 套分配账本 + 1 个争议处理流程。

---

## 二、宏观愿景（压缩版）

- **核心理念**：我为人人、人人为我（互哺逻辑）。
- **社会资本**：将艺术家的名誉、技艺与平台的分配引擎深度绑定。
- **宪法级原则**：分配必须可追溯；分润只来自真实业务；艺术家享有名誉保护与撤回权。

**业务优先条款**：任何治理与分配机制的复杂化，必须以「已跑通的真实订单收入」为前提；未跑通前，贡献只记账不兑现，或只兑现固定小额奖励。

## 三、核心产品与首发策略

### 3.1 数字身份授权（DIL）

- **产品形态**：定制化数字艺术内容（签名、印记、标识）+ 明确边界的使用授权 + 溯源证书（D-Certificate）。
- **价值主张**：为追求个性表达的消费者提供高性价比的「身份印记」；为艺术家提供版权受保护、利益可追溯的全球分发渠道。
- **不涉及**：资产升值、二级市场、投资或收益承诺。

### 3.2 首发产品路径

| 阶段 | 形态 | 说明 |
|------|------|------|
| **Phase 1** | 纯数字/授权类 | 零物流、可标准化、可立即全球交付、100% 对应版权许可费 |
| **Phase 1.5** | 数字+限定实体 | 美国法人统一采购、国内签样、美国分装；实体为「授权履约载体」 |
| **Phase 2** | 开放更多实体 SKU | 定制书法、匾额、空间陈设等 |

**原则**：数字是「铸币权」，实体是「印钞机」。

### 3.3 0 元入口与升级路径（获客与转化核心）

| 层级 | 内容 | 价格带 |
|------|------|--------|
| **Free Tier** | Identity Draft：横版手写签名模板（不可打印、不可实体化、不附带 D-Certificate） | $0 |
| **A 正式数字授权** | 去水印、专属微调、D-Certificate | $99–$199 |
| **B 实体交付** | 原签名风格实体化、材质选择、编号、扫描存证 | $499 起 |
| **C 艺术家定制** | 发起定制请求，报价由艺术家给出 | $1,000+ / $5,000+ |

**阀门**：模板库有限；免费不接受反复修改；免费 ≠ 可扩散资产。

## 四、顶层架构：三权分立与分阶段治理

### 4.1 三层治理结构

| 层 | 主体 | 职责 |
|----|------|------|
| **执行权** | Founder | 产品节奏、技术架构、商业模式；72h 紧急暂停权；**不能修改已生效的分配规则** |
| **自治权** | Domain Council | 按门类设立；新人准入、内部竞争协调、行业伦理；不碰钱、不碰码、不私授 |
| **分配权** | 参数化引擎 | 硬编码、自动执行、不接受人工干预 |

### 4.2 分阶段治理（避免治理超前于业务）

| 阶段 | 治理形态 |
|------|----------|
| **Phase 0**（0–30 天） | Founder 强治理 + 透明账本；规则由 Founder 发布，账本公开 |
| **Phase 1**（30–90 天） | 引入 2–3 位核心创作者代表 + 1 位财务/法务观察员，审核账本与规则变更 |
| **Phase 2**（90 天+） | 再谈委员会/逆向投票；治理复杂化以订单跑通为前提 |

### 4.3 去中心化仲裁（DAS）

- **Level 1**：1 法官 + 3 陪审（小额、一次性）
- **Level 2**：1 法官 + 5–7 陪审（标准争议）
- **Level 3**：3 法官 + 9 陪审（宪政级争议）
- 成本由败诉方承担；判断正确→Accuracy Score 上升；不惩罚错误判断；全流程可回看。

## 五、经济模型：三张表

### 5.1 订单经济表

| 项目 | 说明 |
|------|------|
| 客单价档位 | $49–$199（数字）；$499 起（实体） |
| GMV | 用户支付总额 |
| Direct Costs | 支付手续费、退款准备金、生产/物流 |
| Net Pool | GMV - Direct Costs |
| **艺术家利润部分** | 65% |
| **Anchor Pool（创世公民）** | 5% |
| **公共运营池** | 20% |
| **风险储备金（FCR）** | 10%（含 1% FX Buffer） |
| 退款/争议 | 由 FCR 承担；恶意滥诉可追加成本 |

### 5.2 创世公民 特殊机制

- **Floor% Bucket**：Net Pool 的 5%
- **Cap**：单人单月 Floor 收益 ≤ 过去 6 个月平均分润的 200%
- **Catch-up**：超出部分 50%→FCR，50%→公共运营池
- **Time & Volume Decay**：18 个月线性衰减至 50%；订单 1k/10k 触发阶梯式衰减

### 5.3 贡献记账表（A/B 类）

| 类型 | 内容 | 兑现规则 |
|------|------|----------|
| **A 类** | 直接带来订单的 referral、完成交付、客诉处理成功 | 可参与分润 |
| **B 类** | 发帖、评审、建议、内容传播 | 只记账，延迟兑现；设每月上限防刷 |

### 5.4 批次权益表（版本锁定）

| 批次 | 权益 | 防修改 |
|------|------|--------|
| **创世公民** | 5% Anchor Pool、起草委员会席位、Kill Switch | 写入协议「版本与变更条款」；变更需记录、可回溯 |
| **后续批次** | 无 Anchor Pool 或递减 | 同上 |

## 六、准入与社会资本：背书责任制

- **提名制**：新艺术家加入须由现任议会成员或认证艺术家提名。
- **背书责任**：提名人需为新人的交付信用提供分润权质押；新人严重违约时，提名人承担连带补偿。
- **评分体系**：艺术成熟度 40%、授权意识 20%、交付可靠性 20%、生态多样性 20%；≥75 通过，60–74 观察期，<60 拒绝（6 个月冷却）。

---

## 七、跨境合规与结算：First Dollar to China

- **款项性质**：严格界定为「版权许可费」或「数字内容服务费」；非投资、非分红、非股权。
- **收款主体自由**：支持个人或公司、多地区账户；自行承担税务与合规；平台仅按规则付款。
- **Invoice-on-Demand**：企业收款或法律要求时提供发票/对账单。
- **主权保障**：Kill Switch（一键撤回未来授权）；主版权始终归属艺术家。

## 八、冷启动与 30 天闪电闭环

### 8.1 Anchor Tenant 策略

- **对象**：5–10 位在中国自媒体已获成功的艺术家（自带流量、成熟交付、市场验证）。
- **平台提供**：出海渠道、标准化履约、可追溯分配。
- **要求**：交付 SLA（72 小时内数字交付）、作品范围、授权边界。

### 8.2 72 小时交付 SOP（摘要）

| 阶段 | 时间 | 责任 |
|------|------|------|
| 订单确认 | T+0 | 系统 |
| 授权任务生成 | T+2h | 平台 |
| 艺术家交付 | T+48h | 艺术家 |
| 合规审核 | T+60h | 平台 |
| 用户交付 | T+72h | 平台 |
| 结算触发 | T+72h 后 | 系统（进入 14 天 Escrow） |

### 8.3 营销与转化

- **内容**：利用艺术家已有的英文名视频素材，统一结构（身份钩子 + 人格表达 + CTA），不出现价格。
- **落点**：唯一落点为 Free Identity Draft（0 元签名）。
- **首阶段 KPI**：Landing→Free Draft 完成率 ≥25%；Free Draft 停留 ≥60 秒 ≥40%；点击升级 ≥8–12%。

### 8.4 30 天执行清单

1. **[1–7 日]** 锁定 Anchor：定向沟通 5–10 位头部自媒体艺术家，签署意向加盟书。
2. **[8–15 日]** 建立数字工厂：确定首批 3 个 SKU（签名、个人 Logo、数字印章），制定交付 SOP。
3. **[16–25 日]** 合规渠道搭建：美国实体 Stripe 接口，版权授权合同模板。
4. **[26–30 日]** 压力内测：Founder 真实下单，跑通支付→账本分配→艺术家收款全链路。

## 九、风险清单与兜底

| 风险 | 兜底 |
|------|------|
| R1 供给端质量不可控 | 首批只收可标准化交付者；试单/样稿/SLA |
| R2 消费者不买账 | 首发用礼品场景：毕业、升职、婚礼、移民 |
| R3 退款/纠纷压垮 | 明确可退/不可退边界；分阶段确认；FCR+仲裁 |
| R4 贡献值内斗 | A/B 类贡献；B 类延迟兑现；每月上限 |
| R5 跨境履约复杂 | 先数字；实体只做 1 SKU、1 仓、1 国 |
| R6 IT 过于乐观 | 先用现成工具拼；分配用手工对账跑 50 单 |
| R7 大咖信任成本 | 名誉保护协议；平台问题时可一键下架获补偿 |
| R8 治理公地 | 代码/劳动即分润；纯讨论只有 B 类 |
| R9 合规（类股权） | 分润仅来自真实业务；劳务/版权许可合同 |

## 十、结语

WhoAmI.Art 正在构建的是一套可以承载判断力、信用与长期协作的制度机器。通过结构化设计，让正确的商业行为变得更便宜，让错误的投机行为变得更昂贵。

**核心验证指标**：第一笔跨境付款能否合法、安全地到达中国艺术家账户——这比 Landing Page 和 SKU 重要 10 倍。
