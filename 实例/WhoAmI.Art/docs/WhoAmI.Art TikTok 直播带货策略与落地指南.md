## 一、渠道与双轨策略

### 1.1 TikTok Shop 定位

| 维度 | 说明 |
|------|------|
| **角色** | 收割机：销售「刷到即想拥有」的低单价、标准化艺术品 |
| **SKU** | 现货、预生产、数字授权；不适合需艺术家手写、刻录的定制 |
| **履约** | 48 小时发货要求；定制走 whoami.art 独立站 |
| **分配** | 按**净额**（平台抽成后）应用 65/5/20/10 |

### 1.2 渠道收入分配原则

> 凡经由第三方平台产生的交易，以**平台扣除佣金、手续费、达人佣金等后的实收净额**为分配基数，按宪章约定的比例（65/5/20/10）进行分配。渠道成本由各参与方按分配比例共同承担。

### 1.3 订单归集

- 多渠道订单可导入 Shopify 或统一 OMS
- 分配引擎（65/5/20/10）独立于 Shopify，对净额执行

## 三、MVP 方案：微信视频 + 美国主持人

### 3.1 技术链路

```
中国艺术家（微信视频，境内可用，无需 VPN）
       │
       ▼
美国主持人电脑（WeChat 接收 + OBS 捕获 + 推流到 TikTok）
       │
       ▼
TikTok Live（原生推流，合规）
```

### 3.2 IT 部门职责

| 工作 | 说明 |
|------|------|
| **推流环境** | 电脑、摄像头、麦克风、上行 ≥10 Mbps |
| **软件** | WeChat 桌面版、OBS Studio、TikTok Creator 配置 |
| **场景预设** | 主持人全屏、嘉宾全屏、画中画 |
| **SOP** | 开播清单、主持人操作手册、艺术家指引、故障预案 |
| **测试** | 中美联调、画质/码率、延迟、TikTok 审核 |

**MVP 预估**：3–5 个工作日可跑通首场。

### 3.3 带宽与直播间数量

| 专线带宽 | 支持直播间数 |
|----------|--------------|
| 10 Mbps | 1–2 间 |
| 20 Mbps | 3–4 间 |
| 50 Mbps | 8–10 间 |
| 100 Mbps | 15–18 间 |

*单间按 4–5 Mbps 上行预留。*

## 四、主持人机制

### 4.1 角色定位

| 角色 | 来源 | 职责 |
|------|------|------|
| **主持人** | 在美国的华人 | 翻译、控场、OBS 推流、TikTok 直播 |
| **艺术家** | 中国 | 创作展示、现场书写、中文回应 |

### 4.2 艺术家不会英文时的互动

| 策略 | 做法 |
|------|------|
| **主持人实时翻译** | 观众→主持人→艺术家；艺术家→主持人→观众 |
| **创作即互动** | 观众留名字→艺术家现场写；技法展示无需语言 |
| **5–10 句英文** | Thank you, Hello, Beautiful 等，艺术家可背熟 |
| **非语言** | 微笑、点头、竖大拇指、展示作品 |

### 4.3 结算关系

| 原则 | 说明 |
|------|------|
| **平台不介入** | 艺术家与主持人私下约定费用（按时长/按订单/按场次等） |
| **艺术家自付** | 艺术家从 65% 中自行支付主持人，结算方式自定 |
| **平台不分成** | 平台不从中抽成，不代扣代付 |

### 4.4 主持人可换

主持人按需更换，艺术家与主持人的合作条件由双方自行商定，平台不参与。

## 五、账号控制权

| 原则 | 说明 |
|------|------|
| **账号归属** | 平台或平台+艺术家拥有 TikTok 直播账号 |
| **价值沉淀** | 粉丝、内容历史、口碑归账号，不随主持人流失 |
| **主持人权限** | 限时、限定用途（仅直播）；不得修改密码、绑定、注销 |
| **协议约束** | 主持人签署：无所有权主张、保密、离开时交还权限 |

## 六、直播中心（SST 后可选）

### 6.1 模式

- 平台投资集中直播场地，艺术家到中心轮班
- 专线覆盖，艺术家本人直推 TikTok，效果最优

### 6.2 治理与成本约束

| 原则 | 做法 |
|------|------|
| **非强制** | 中心为可选服务，艺术家可自选在家/他处 |
| **SST 后投入** | 验证期不建，达日 200 单后再考虑 |
| **先租后建** | 优先租用现有直播基地 |
| **成本占比** | 中心支出占公共运营池设上限 |

### 6.3 专线获取

- 中国电信/联通/移动 政企专线，或授权代理商
- 用途表述：「跨境电商直播」「海外营销」
- 费用区间：约 5–20 万/年（依带宽）

## 七、中美数据与媒体架构

### 7.1 数据流（中国艺术家不依赖 VPN）

| 场景 | 方案 |
|------|------|
| **订单、后台** | 双区部署：中国门户在阿里云/腾讯云，美国侧 Shopify+分配引擎 |
| **图片/视频上传** | 阿里云 OSS / 腾讯云 COS，境内 ingest |
| **直播** | 阿里云直播 / 声网 Agora，境内推流，全球 CDN 分发 |

### 7.2 微信为何无需 VPN

微信在中国有完整基础设施，中国用户只访问腾讯境内服务器；跨境由腾讯自建骨干完成，用户不直连境外被封服务。

## 八、产品与入口（关联讨论）

### 8.1 意图导向入口

- 按意图不按品类：「铭刻自我」「寄托情感」「支持源头」
- 首页不做货架，做决策隧道；品类收在意图路径内

### 8.2 策展机制

- AI 层：意图翻译、参数化
- 人工层：高单价（如 $1000+）时人工策展节点介入
