# 人际关系档案系统
**用途**：Founder 的人际关系外挂。当你在对话中提及某人时，AI 会将信息保存到对应人物档案。

---

## 数据结构说明

| 文件 | 用途 |
|------|------|
| `01_索引.md` | 总览表，按关系类型/姓名快速查找 |
| `人物/[姓名].md` | 单人档案，每次提及此人时在此追加 |

## 单人档案结构（每人的 .md 文件）

```markdown
# [姓名]
- 关系类型：合作伙伴 | 投资人 | 家人 | 朋友 | 同事 | GAIA相关 | 其他
- 首次记录：YYYY-MM-DD
- 最后更新：YYYY-MM-DD

## 背景速写
（身份、机构、与你的交集）

## 互动记录（倒序，最新在前）
- YYYY-MM-DD：[事件/对话要点]
- YYYY-MM-DD：...

## 待办/提醒
- [ ] 

## 备注
```

## 使用方式

1. **主动保存**：说「把刚才说的关于 XX 的记下来」→ AI 会更新 `人物/XX.md`
2. **自动捕获**：当你提到某人 + 具体信息时，AI 会询问「是否记入 XX 档案？」
3. **查询**：说「XX 是谁」「我和 XX 上次聊了什么」→ AI 读取对应档案回答
4. **幕僚建议**：说「下周要见 XX，帮我准备一下」→ AI 综合档案与 GAIA 背景给出建议

## 保密说明

- 本文件夹仅在本地，不随代码仓库公开（可加入 .gitignore）
- 建议不在此存储密码、账号等敏感信息
