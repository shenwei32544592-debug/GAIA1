## 一、GAIA 是什么 / 要做什么

**GAIA**（Global Art & Identity Alliance，全球艺术与身份联盟）是为原创者、以及为在 AI 时代追问「我是谁」的每一个人而存在的**全球公共体系**。它不是公司、不是商店、不是某一国机构，而是一套**开放的规则与技术体系**。

**终极目标**：建立**艺术家的理想秩序**。在这一秩序下：
- 创意主权被尊重
- 源头被铭记（署名不可抹除）
- 收益可回流（65% 归艺术家）
- 共识有力量（同行裁定，非平台独家裁决）

今日从签名艺术切入（signa.ink 为 001 号实例），未来可覆盖绘画、雕塑、手工艺等更多形式。

---

## 二、为什么存在

当前体系对艺术家的系统性失效：

| 问题 | 现状 |
|------|------|
| **源头被遗忘** | 创作者在传播与商业化中被抹除，技法被模仿，本人从历史与收益中消失 |
| **正义被垄断** | 「谁是原创者」由大平台、法律成本、话语权决定；真理常输给资源 |
| **尊严被消解** | 艺术沦为文件与像素，在现实世界无痕 |

GAIA 建立一套文明级协议：保护创意主权、署名不可篡改、收益可追溯、争议由同行共识裁定。

**历史类比**：保护艺术家的原创，其威力等同于当年专利保护对全人类技术发展造成的革命性变化。国家层面的法规、登记、归档是文明的物理基础；有章可循，非突发奇想。现有全球治理在发现、保护、发扬软创上存在制度欠缺；唯有底层联系与全球协作可突破壁垒。「高手在民间」——创造者从未缺席，但制度尚未承载。这是时代的呼唤，也是历史的命题。详见《GAIA 成立理由书（艺术家为何联合）》§二点五。

## 五、核心术语（简要）

| 术语 | 含义 |
|------|------|
| **GAIA** | 艺术家联盟；文明协议与联盟组织，非公司 |
| **节点 Node** | 体系中所有参与者（人/组织/公益体）；不称「用户」 |
| **创世者 Founder** | 系统发起人；执掌执行权；权力随时间衰减 |
| **裁定者 Steward** | 参与共识裁定的专业节点；执掌裁定权，不制定规则 |
| **创世公民 Batch 0** | 首批签《自然人宪章》者；制宪会议代表 |
| **FCR** | 创始贡献者权利；建「国」期贡献者的分配权与署名；非股权 |
| **创意载体规范** | 物理载体必须臣服于精神印记；工业基底 ≥0.15mm；自由源头严禁算法平滑 |
| **三体架构** | 源头母本 + 工业分身 + 数字分身 |
| **分润 65/5/20/10** | 艺术家 65%；创世 5%；运营 20%；风险储备 10% |
| **PAC 公众指认** | 任何人可提议纠正署名；经同行裁定生效 |

## 附录 A：不证自明的事实（成立理由书 §二）

| 事实 | 含义 |
|------|------|
| 创意主权先于传播 | 原创行为自产生之时起，即构成主权行使 |
| 源头应被铭记 | 创意被复制、传播、学习，源头仍应被记录与尊重 |
| 传播不应吞噬原创 | 商业化可以发生，但不能抹除创造者 |
| 共识应具有力量 | 当足够多的同行与公众形成判断，这种判断本身应当具备约束力 |
| 尊严优先于效率 | 在效率与原创者尊严冲突时，应优先保护尊严 |
| 价值的产生者应拥有规则表决权 | 谁创造价值，谁应参与决定分配规则 |

## 附录 B：GAIA 如何运作（公共释义版摘要）

- **源头登记**：创作者申报原创技法或作品；留下不可篡改的时间与署名。
- **公众指认与同行裁定**：任何人可发起源头指认；由专业人士评议，形成公开、可追溯的结论。
- **长期反哺**：创意被持续使用时，收益自动流向源头；亦可导向公益、教育。

**GAIA 不是什么**：不是投机项目、不是加密货币噱头、不是艺术评级机构、不是中心化平台。GAIA 是一套规则、一种共识、一种长期约定。
