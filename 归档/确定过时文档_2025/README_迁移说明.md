# 文件迁移说明

## 如何完成文件迁移

由于 Windows PowerShell 对中文路径存在编码问题，请手动执行迁移脚本：

1. 打开 **CMD** 或 **Cursor 集成终端**（建议使用 CMD，编码更稳定）
2. 进入项目根目录：
   ```
   cd /d d:\GAIA_System_OS
   ```
3. 执行脚本：
   ```
   python "00_索引与导航\迁移脚本_执行此文件移动.py"
   ```

执行成功后，根目录下的 51 个文件将按分类移动到 01～06 子目录及归档目录。

## 迁移后的目录结构

```
GAIA_System_OS/
├── 00_索引与导航/     ← 快速查找表、Master Index、更新日志
├── 01_宪法与治理/
├── 02_协议与规范/
├── 03_创世与权益/
├── 04_工程与落地/
├── 05_对外与外交/
├── 06_解释与叙事/
├── 人际关系/
└── 归档/
```

## 若脚本报错

检查 Python 是否正确安装，或尝试在 Cursor 中右键 `迁移脚本_执行此文件移动.py` →「在终端中运行 Python 文件」。
