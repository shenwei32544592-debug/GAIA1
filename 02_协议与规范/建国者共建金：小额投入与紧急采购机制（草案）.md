# 建国者共建金：小额投入与紧急采购机制（草案）

**用途**：建国者以小额金钱投入表达核心支持与相互信任；通过财务透明示范建立「无人贪污」的可验证共识；为无节点能出力但须尽快完成的任务提供**对外采购**资金  
**版本**：草案 v0.1  
**依据**：《创世贡献记录与确认机制》《公益金池细则》《基建期艺术家共投机制》；成立理由书「重投入即护城河」「共同自投机制」  
**原则**：自愿、小额、透明、专款专用；信任示范优先于募资规模

---

## 一、核心逻辑

| 目的 | 设计 |
|------|------|
| **支持与信任表达** | 建国者除精力时间外，以小额金钱投入表明「真金白银在赌桌上」；相互信任的信号 |
| **透明示范** | 每笔入账、出账、用途全可查；成果可见；建立「无人贪污」的可验证共识 |
| **加速核心任务** | 当建"国"任务清单中某项**无节点能出力**但**必须尽快完成**时，从此池对外采购 |
| **低门槛、高参与** | 单次可少至 100 元；不设硬性频率；参与即表明态度 |

---

## 二、参与资格与投入方式

### 2.1 参与资格

| 对象 | 说明 |
|------|------|
| **建国者** | 创世者、创世技术官、创世公民、已签 FCR 者、论坛认领/附和≥1 次者；具体由创世期书面约定 |
| **自愿** | 不参与不歧视；参与即表明对透明机制与建国进度的支持 |

### 2.2 投入方式

| 项目 | 约定 |
|------|------|
| **单次金额** | 建议起档 100 元/次；可自愿多捐；不设上限 |
| **频率** | 自愿；可每月/每双周/按需；不强制 |
| **渠道** | 约定账户（见 §四）；入账时标注参与人、日期、金额 |
| **性质** | 自愿共建；不构成投资；无回报承诺；不产生治理权或 FCR 比例 |

---

## 三、用途与动用规则

### 3.1 专项用途

**仅用于**：建"国"任务清单中，**无节点能出力**且**须尽快完成**的任务之外部采购。

| 可动用情形 | 示例 |
|------------|------|
| 技术模块采购 | 某组件无内部 IT 能开发，须外包 |
| 合规/法务采购 | 跨境合规咨询、协议存证服务 |
| 设计/品牌采购 | 官网设计、品牌物料，无内部资源 |
| 实物采购 | 001 套件打样、档案盒等，无内部供应链 |

**不可动用**：日常运营、个人消费、与建"国"任务无关之用途。

### 3.2 动用程序

| 环节 | 要求 |
|------|------|
| **提议** | 创世者或任务责任人发帖：任务 X 需采购 Y；预算 Z；无节点可承担；请动用共建金 |
| **确认** | 创世技术官（或指定确认人）书面确认：该任务确属清单、确无内部资源可承担 |
| **决议** | 创世者签署拨款决议：用途、金额、收款方 |
| **披露** | 决议与划款记录全节点可查；论坛或看板公示 |

### 3.3 透明度要求

| 项目 | 要求 |
|------|------|
| **入账** | 每笔标注：参与人、日期、金额；实时或日更新 |
| **出账** | 每笔标注：用途、金额、收款方、决议 ID、日期 |
| **余额** | 期初 + 入账 - 出账 = 期末；全节点可查 |
| **公示** | 建议在论坛「建国者共建金」专帖或看板中持续更新 |

**原则**：阳光下的财务；哪怕 1 元也可追溯；建立「无人贪污」的可验证共识。

---

## 四、账户与存放

| 项目 | 约定 |
|------|------|
| **存放** | 独立银行账户或与既有联盟账户分科目；与公益金池、建国债券、基建池**分离** |
| **托管** | 创世者或指定执行节点托管；须有双人复核或约定流程 |
| **审计** | 可纳入年度审计；任何参与者可提请抽样复核 |

---

## 五、与既有机制的关系

| 机制 | 区别 |
|------|------|
| **公益金池** | 公益金池用于公益、教育、传承；共建金用于建国期紧急采购；二者独立 |
| **建国债券** | 债券为融资工具、到期还本；共建金为自愿共建、无回报承诺；可并存 |
| **基建池 / 艺术家共投** | 基建池来自艺术家分润让渡、有返还规则；共建金来自建国者直接现金投入、无返还 |
| **建国回补** | 回补来自后进艺术家；共建金来自建国者自身 |

共建金为**建国期专用、建国者专用**；制度成熟、平台有现金流后，可经共识决定是否延续或转入其它池子。

### 5.1 互信驱动与成功信号（战略扩展）

当合作机制中**互信锁定**且出现**明确的成功信号**（首单闭环、首笔跨境、FCR 流水、订单爬坡等）时：

| 参与方 | 可能行为 | 机制归属 |
|--------|----------|----------|
| **艺术家** | 愿将所赚大部分**借给**联盟用于快速发展 | 基建期艺术家共投（利润预借）；须有还款条件、顺序、期限 |
| **供应商** | 愿接受账期延长或预付款延迟，换取长期合作 | 可经共识扩展为供应商协作池 |
| **协作伙伴** | 愿将早期收益再投入联盟 | 可经共识扩展；与共建金、基建池并列 |

**重要节点优先认购**：发展基建基金**优先接受**艺术家、供应商、类帐户的投资认购；其与系统成功高度相关，有利于合作机制发展更平稳、更快速。*详见《建国方案》§3.5、《基建期艺术家共投机制》*

**原则**：借出与分润分离；借出**不产生治理权或永久统治权**；须有可审计的还款规则。  
*配套：《联盟胚胎发展》§0.4 简化造血路径与互信驱动收益再借入、《基建期艺术家共投机制》*

## 六、合规口径

| 表述 | 用途 |
|------|------|
| 建国者共建金 / Founders' Mutual Build Fund | 池名称 |
| 自愿共建、小额支持 | 参与性质 |
