📜 全球艺术联盟（GAIA）

创世者行动纲要（Founders’ Action Charter · v1.0）

序言（Purpose）

我们认为以下事实是清晰且不可回避的：

  1. 人类创造力的产生是分散的、不可预测的、往往发生在边缘地带的；
  2. 现有的艺术市场与版权体系，无法持续识别、保护和回馈“源头级创见”；
  3. 技术的发展正在加速创意的传播，却没有同步建立公平的溯源与回馈机制。

因此，原创者的尊严正在系统性流失。

我们并不试图修补旧制度，也不试图用情绪对抗现实。
我们选择建立一套新的、可执行的协议体系，使原创能够在传播中持续获得尊重、记录与回馈。

这套协议的第一个实例，称为 signa.ink；
其最终汇聚的上层结构，称为 全球艺术联盟（GAIA）。

第一部分：不可妥协的基本原则（Foundational Principles）

我们确认以下原则将作为一切系统设计、产品决策与治理机制的最高约束。

1. 原创即主权（Originality as Sovereignty）

任何可识别的原始创见——无论其形式是签名、技法、材料逻辑还是工艺流程——
其源头个体或群体，应被视为该创见的主权拥有者。

该主权不因传播、模仿、教学或商业化而自动消失。

2. 传播不应消灭源头（Propagation Shall Not Erase Origin）

一个创意被全球学习和使用，不应意味着其原创者被历史抹去。

系统必须具备：

  • 溯源能力
  • 共识确认能力
  • 长期回馈能力

否则，传播本身就是一种结构性剥夺。

3. 正义应优先于效率（Justice Before Efficiency）

当效率与公平发生冲突时，
系统必须选择可解释的公平，而非不可见的效率。

任何为了“规模”“增长”而牺牲原创尊严的系统设计，
都将被视为违背本纲要。

4. 正义不依赖中心裁决（Justice Without Central Judges）

在艺术领域，真正的判断权应来源于：

  • 同行的专业共识
  • 长期的公共认知
  • 可追溯的历史证据

而非单一平台、资本或司法系统的垄断裁定。

第二部分：我们所面对的问题（Statement of Problems）

我们明确指出，当前系统性问题包括但不限于：

  1. 源头创作者的不可见性
    大量原创技法与艺术体系的发起者，无法在市场中留下可持续的身份与收益。
  2. 贡献与回报的断裂
    创意价值在传播链条中被不断放大，但回报机制仅在末端分配。
  3. 审美与权力的中心化
    审美判断被算法、平台或少数机构固化，缺乏同行共识机制。
  4. 公益与艺术价值的脱钩
    艺术创造难以自然转化为对社会与文化的长期贡献。

这些问题并非道德失误，而是系统缺失。

第三部分：我们的解决路径（Action Framework）

我们选择一条分阶段、可审计、可修正的发展路径。

第一阶段：创世实例（signa.ink）

  • 以“签名”这一最基础、最具身份象征意义的创作形式为切入口；
  • 建立：
    • 原创提交
    • 工业可行性校验
    • 物理交付
    • 原子化分润

signa.ink 的角色是：
验证这套协议在现实世界中可以运行。

第二阶段：协议宪法化（GAIA Core）

在实例验证后，逐步冻结以下协议层：

  1. 身份协议：统一的创作者与参与者身份模型
  2. 资产协议：跨媒介、跨平台的原创资产抽象
  3. 共识协议：同行评审与公众指认机制
  4. 金流协议：递归式、可追溯的价值回馈

GAIA 不作为商业主体存在，而作为跨平台可接入的协议层存在。

第三阶段：联盟化与公益闭环

  • 允许多个艺术平台（绘画、雕塑、工艺等）接入 GAIA；
  • 允许原创收益的一部分，永久指向公益与文化保护节点；
  • 形成一个无需强制、但持续运转的社会贡献网络。

第四部分：关于源头保护的特别声明（On Origin Justice）

我们郑重确认：

  1. 原创保护不以“自我申报”为唯一前提；
  2. 系统必须允许公众与同行主动指认被忽视的源头创作者；
  3. 当长期共识形成时，系统应具备：
    • 追溯确认原创身份
    • 锚定其历史地位
    • 调整未来收益流向

即便该原创者：

  • 地处偏远
  • 缺乏市场能力
  • 已不在世

其创见仍应在系统中获得位置。

第五部分：创世者的自我约束（Founders’ Commitments）

作为该体系的发起者，我们确认以下自我约束将长期有效：

  1. 我们不拥有艺术的解释权，只维护协议的执行权；
  2. 我们不垄断入口，所有接口均需具备未来外部接入可能；
  3. 我们不追求短期商业最大化，以牺牲原创尊严为代价；
  4. 我们接受未来世代通过共识机制对本体系进行修正。

结语（Closing）

我们并不宣称自己能够解决艺术世界的所有问题。

但我们确信：
**如果连“源头应被尊重”这一点都无法被系统化实现，
**那么任何繁荣都只是暂时的。

此纲要并非终点，
而是一个明确的起点。

**我们选择站在原创者这一侧，
**并愿意为此承担长期的系统责任。
