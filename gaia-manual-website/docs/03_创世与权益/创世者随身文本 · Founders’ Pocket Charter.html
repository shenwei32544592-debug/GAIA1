📜 全球艺术联盟（GAIA）

创世者随身文本 · Founders’ Pocket Charter

一、我们为什么存在（WHY）

我们确认一个长期被忽视的事实：

世界并不缺艺术，缺的是对“源头”的尊重。

当创意被无限复制、模仿、传播时，
真正的原创者却往往被市场、平台和历史抹去。

这是系统性不公，而非个体失败。

二、我们要解决什么（WHAT）

我们不是要建立一个更大的艺术市场，
而是要建立一套可执行的“原创主权协议”，使：

  • 原创可以被溯源
  • 贡献可以被记录
  • 价值可以被长期回馈
  • 尊严可以被系统性保护

三、我们的基本立场（NON-NEGOTIABLE PRINCIPLES）

  1. 原创即主权
    原始创见属于源头个体，不因传播而失效。
  2. 传播不应消灭源头
    世界可以学习你，但系统必须记住你。
  3. 正义优先于效率
    当规模与公平冲突时，我们选择公平。
  4. 共识高于裁决
    艺术正义应来自同行与公众，而非中心权力。

四、我们的路径（HOW）

第一步：signa.ink
以“签名”为起点，验证原创 → 工业 → 交付 → 回馈的完整闭环。

第二步：协议宪法化（GAIA Core）
冻结身份、资产、共识、金流四大协议层，允许跨平台接入。

第三步：全球联盟与公益闭环
让艺术收益自然流向社会、文化与被忽视的源头创作者。

五、关于源头保护的底线声明

  • 原创不以“自我申报”为唯一条件
  • 公众与同行有权主动指认被忽视的源头
  • 一旦共识成立，系统必须：
    • 锚定其历史身份
    • 调整未来收益流向
    • 即便其已不在世

正义可以迟到，但不能缺席。

六、创世者自我约束（OUR LIMITS）

我们承诺：

  • 不垄断审美解释权
  • 不牺牲原创尊严换增长
  • 不封死未来修正空间
  • 接受后来者通过共识修订系统

七、我们要留下什么（LEGACY）

**不是一家成功的公司，
**而是一套长期有效的公平机制。

如果多年以后，
一个偏远地区的创作者，
因早年的一个创见，
仍能被世界记住、被系统回馈——

那么，这一切就是值得的。

— GAIA 创世者记录文本 · v1.0
