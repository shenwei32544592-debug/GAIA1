# 基建期艺术家共投机制（草案）

**全称**：联盟基建池与艺术家自愿共投细则  
**用途**：定义艺术家在完全自愿前提下，将部分早期分润暂时转化为公共基础设施投入、并按规则获得优先返还的机制；实现**内生融资能力**，降低对外部资本依赖  
**版本**：草案 v0.1  
**性质**：协议附件，与《分润与财务逻辑补充》配套；须经共识程序通过后生效  
**依据**：联盟宪章关于分配透明、取之于民用之于民之原则；成立理由书关于「基建经济学」「重投入即护城河」之论证

---

## 一、核心定位

**GAIA 并不依赖外部资本才能启动。**

在制度设计上，联盟允许艺术家在**完全自愿**的前提下，将部分早期分润暂时转化为公共基础设施投入。这不是牺牲，而是一种基于透明账本与自动返还规则的**共同自投机制**（Endogenous Financing）。

由于联盟基础设施将显著放大艺术作品的长期定价能力，这种投入并非零和博弈，而是对未来现金流的提前布局。当系统规模达到既定阈值，这些投入将依据可审计规则被优先返还。

**成功信号效应**：当合作机制中互信锁定且出现**明确的成功信号**（首单闭环、首笔跨境、FCR 流水、订单爬坡等）时，艺术家愿将所赚大部分借给联盟用于快速发展的意愿显著提升。此时参与率与借出比例可高于基准预期。*配套：《联盟胚胎发展》§0.4、《建国者共建金》§5.1*

---

## 二、四条硬约束（不可突破）

| 约束 | 内容 |
|------|------|
| **禁止模糊** | 必须明确比例、期限、触发条件、最大损失边界；禁止「大家先少拿点」「以后肯定会赚回来」等模糊表述 |
| **禁止永久性让利** | 任何暂时借给联盟的分润，必须有可返还路径；返还期可长，但不能无期限 |
| **禁止道德绑架** | 艺术家永远可选择不参与；不参与 ≠ 被歧视；参与者仅获得更高长期回报权重 |
| **禁止治理权膨胀** | 共投不产生投票权、控制权；艺术家是**债权贡献者**，非股东、非投资人 |

## 三、机制设计（示意参数，待共识确定）

### 3.1 参与方式

| 方式 | 说明 | 性质 |
|------|------|------|
| **方式 A：分润弹性** | 艺术家自愿在基建期将其分润比例从 65% 暂时调为 55%（或 60%）；差额 10%（或 5%）自动进入【联盟基建池】 | 可选、分批、可随时退出（退出后不再享有返还权） |
| **方式 B：利润预借** | 艺术家自愿将当笔/当期部分利润「借给」联盟；系统记录为【基建债权/创世积分】 | 可选、单笔上限、总额上限 |

**参与入口**：系统提供明确的「加入基建共投」勾选项；默认不勾选；艺术家须主动选择进入。

**重要节点优先认购**：发展基建基金（基建池）**优先接受**重要节点人的投资认购——艺术家、供应商、类帐户（机构/基金等合规主体）。其与系统成功高度相关、协调成本低，有利于合作机制发展更平稳、更快速。供应商、类帐户的参与方式与返还规则，可经共识扩展，与艺术家共投同池或分池管理。详见《建国方案》§3.5。

### 3.2 基建池用途

| 用途 | 说明 |
|------|------|
| **IT 基础设施** | 分配引擎、RuleID、溯源系统、跨币种结算等；自研与采购须符合[《GAIA IT 供应链准入与 API 隔离标准》](../04_工程与落地/GAIA%20IT%20供应链准入与%20API%20隔离标准.md) |
| **合规与法务** | 跨境合规、协议存证、争议处理机制 |
| **品牌与获客** | 初期品牌建设、策展节点对接、海外渠道拓展 |

**资金去向全平台可查、可审计。** 不得用于与基建无关之用途。

### 3.3 返还规则（规则触发，非人决定）

| 项目 | 约定（示意） |
|------|--------------|
| **触发条件** | 系统达到既定阈值（如：累计订单数 ≥ N；或 GMV ≥ $X；或加盟艺术家数 ≥ M） |
| **返还倍数** | 历史让利值 × 约定倍数（如 1.5x 或 2x；上限封顶，避免不可持续） |
| **返还顺序** | 按参与时间先后，或按让利额加权；先参与先返还 |
| **返还形式** | 从后续订单分润中优先划拨；或从联盟基建池结余中划拨 |

**返还是规则触发，非任何人临时决定。** 触发条件、倍数、上限须写入协议，全平台可查。

### 3.4 最大损失边界

| 项目 | 约定 |
|------|------|
| **单人参与上限** | 单艺术家累计让利额 ≤ 其过去 12 个月分润总额的 X%（如 20%）；防止过度暴露 |
| **总池上限** | 基建池累计募集额 ≤ 约定上限；超限不再接受新进入 |
| **退出机制** | 参与者可申请退出；退出后未返还部分按约定处理（如：放弃返还权，或按比例延后） |

## 四、与现有池子的关系

| 池子 | 与基建共投的关系 |
|------|------------------|
| **艺术家利润部分（65%）** | 共投之来源为艺术家自愿从 65% 中让渡的部分；非强制 |
| **公共运营池（20%）** | 基建池独立于 20%；20% 用于运营成本，基建池用于基建投入 |
| **公益金池** | 公益金池用于公益；基建池用于基建；二者分离 |

基建池为**独立会计科目**，与 65/5/20/10 主分配结构并列，但不改变主结构。

## 五、金融类比（供理解）

本机制在金融上近似：

- **合作社资本账户（Patronage Capital）**：成员让渡当期收益，换取未来按贡献比例返还
- **可回收的协议税（Refundable Protocol Fee）**：早期略高比例，规模化后反向返还给早期贡献者
- **无投票权、无控制权的内部债权**：艺术家不获得治理权，仅获得优先返还权

---

## 六、为何比外部资本更安全

| 外部资本 | 艺术家内部共投 |
|----------|----------------|
| 要求控制权 | 不要控制权 |
| 要求退出路径 | 不要退出流动性（与系统长期绑定） |
| 要求规模优先于正义 | 只要系统长期健康 |
| 可能要求修改分配规则 | 与联盟目标完全一致 |

**这是资本结构最干净的一种融资方式。**

## 七、生效条件

1. 本草案经治理共识（如 75% 裁定者或创世公民）通过
2. 具体参数（比例、倍数、触发阈值、上限）经共识确定并写入生效版
3. 系统具备记录、披露、返还之技术能力
