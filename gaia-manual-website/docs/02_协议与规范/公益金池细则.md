# 公益金池（全平台公益积累金）细则

**全称**：公益金池（全平台公益积累金，Public Good Fund Pool）  
**用途**：全平台唯一公益金账户的治理、来源、动用权与披露规则；供财务、治理与审计参考  
**版本**：1.0  
**性质**：协议附件，与《递归反哺与源头确权细则》§5.8 配套；本细则为独立索引与摘要，便于快速查阅

---

## 一、定义与定位

| 项目 | 内容 |
|------|------|
| **全称** | 公益金池（全平台公益积累金，Public Good Fund Pool） |
| **简称** | 公益金池 |
| **定位** | 全平台**唯一**公益金银行账户或公益信托 |
| **原则** | 账户唯一、记录明确、动用权已定、不可能超支、余额永不为负；明细账全平台共享可查 |

---

## 二、入池来源

| 来源 | 说明 | 决策权 |
|------|------|--------|
| **① 1% 名誉成本** | 创世者与创世技术官各认捐 Net Pool 的 0.5%，合计 1% | 创世者提议权与决策权**专属**；创世技术官知情权与建议权、无决定权；其它节点提议权与监督权 |
| **② 行业始祖部分** | 0.1% 先入待认领池，经公投认定无法指认或满 N 年无指认后**转入**公益金池 | 平台级治理（Domain Council 或行业代表联合）决策 |
| **③ 主动捐赠** | 任何人可主动捐赠（含客户、艺术家、联盟内任意角色）；平台须提供捐赠通道 | 按捐赠时约定或默认并入创世者决策权部分或行业始祖部分 |

### 2.1 主动捐赠：通道、勋章、筛选与定向捐赠

| 项目 | 约定 |
|------|------|
| **捐赠通道** | 客户、艺术家、联盟内任意角色——任何人皆有通道可捐；方便大家对公益的认可与参与 |
| **公益勋章** | 系统对捐赠行为授予**公益勋章**等精神奖励；鼓励爱心，不涉金钱回报；勋章持有人享有代表联盟出席公益活动时的**优先参与权** |
| **定向捐赠** | 联盟对接外部或特定公益项目时，可引导捐赠者**指定用途**（如某次灾后重建、某类艺术教育、某非遗传承项目）；款项定向划付，实现公益慈善的精准对接；为联盟声誉与公益事件营销带来天然优势 |

## 三、动用权分层

同一池、同一银行账户；平台账簿按**决策权类型**分设虚拟子余额。

| 类型 | 来源 | 决策主体 | 用途边界 |
|------|------|----------|----------|
| **行业始祖部分** | 从待认领池转入的 0.1% | 平台级治理（Domain Council 或行业代表联合） | 传承、教育、扶持新人、记录；须经平台级拨款决议 |
| **创世者决策权部分** | 1% 名誉成本 | **创世者（Founder）专属** | 公益、教育、行业扶持、新人孵化、记录与传承、联盟内成员人道救助（10% 风险储备金结余不足或暂不可用时的补充）、符合平台使命的其它公益用途；**禁止**个人消费、无关商业投资、政治捐献 |

**联盟内成员**：曾在体系内拥有账户或完成联盟宪章承认行为的自然人/实体；认定以系统可追溯记录为准。详见《GAIA 标准术语与概念表》。

**互不挪用**：平台级决议仅可从行业始祖部分划出；创世者决议仅可从创世者决策权部分划出。

## 四、创世者决策权：约束与程序

### 4.1 权力分层：决策、知情、提议、监督

| 角色 | 权力 | 说明 |
|------|------|------|
| **创世者（Founder）** | 提议权、决策权（专属） | 可提议用途；1% 用途的最终决定；须符合用途边界、受披露与审计约束 |
| **创世技术官** | 知情权、建议权（无决定权） | 拨款前 7 天披露；可提交书面建议；决议须注明是否采纳及理由 |
| **其它节点** | 提议权、监督权 | 可提议用途方向或具体项目；可查阅披露、可对异常提请调查；无决定权、无否决权 |

**协作与落地可行性**：提议权=提案入口（论坛/治理界面）；创世者拥有提议权与决策权，可主动发起或采纳入提议；监督权=全平台可查披露+异常可提请；与现有披露、审计机制一致，无新增实验性依赖。三角制衡（决策/知情建议/提议监督）增强公信力。

### 4.2 CTO 知情权（拨款前 7 天）

| 环节 | 要求 |
|------|------|
| **提前披露** | 创世者须提前 **7 天** 向 CTO 披露用途草案 |
| **CTO 建议** | 7 天内 CTO 可提交**书面建议**；可不提 |
| **决议注明** | 拨款决议须注明：CTO 未提建议，或已听取建议并写明「已采纳」或「未采纳，理由：[说明]」 |

### 4.3 用途边界与约束

| 条款 | 内容 |
|------|------|
| **单笔上限** | 单笔 ≤ 该部分当期余额 30%，或 ≤ 过去 12 个月入账 50%，取较低值；经裁定者或门类理事会事前书面豁免可例外 |
| **大额披露** | 单笔 ≥ 该部分当期余额 20% 时，须在季度披露中**单独列出**用途与决策依据 |
| **年度复核** | 每财年结束后 30 日内，创世者须向治理披露该部分当年拨款汇总 |
| **审计** | 年度审计须覆盖；裁定者或门类理事会可提议抽样复核 |

### 4.3.1 联盟内成员人道救助的专项约束

**目的**：防滥用、减道德压力；将 1% 成员救助限定为 10% 池的补充，并设明确边界。

| 约束 | 内容 |
|------|------|
| **用途收窄** | 仅限**重大人道危机**：重大疾病、自然灾害、意外致重大人身或财产损失；**不含**一般困难（如失业、小额周转、一般生活困难） |
| **前置筛选** | 申请人须已向 10% 风险储备金池提出救助申请，或经确认 10% 池结余不足/暂不可用；**不得**绕过 10% 直接向 1% 申请 |
| **年度比例上限** | 每财年用于联盟内成员人道救助的拨款总额 ≤ 创世者决策权部分**该财年入账**的 **20%**；超限须待下财年 |
| **证明要求** | 须提供可核验材料：医疗证明、灾情证明、意外事故认定等；无证明不予受理 |
| **标准公开** | 受理条件、不受理情形须在官网或治理板块公示；申请人可自行对照，减少无效申请 |

**创世者可委托**：创世者可书面将「联盟内成员人道救助」类决策权委托给联盟内某委员会（如裁定者 + 门类理事会联合）；其它 1% 用途仍由创世者决定。委托须公告并更新决策权归属记录。

### 4.4 决策权移交

**原则**：1% 用途决策权不追求永久个人化；制度成熟后由集体治理更符合联盟精神。创世者可在适当时机自愿交出，或于特殊情形下经程序移交。

| 情形 | 移交方式 | 说明 |
|------|----------|------|
| **死亡、失能、长期失联** | 按《签字权与法人地位承继规则》或专门约定，移交给指定继任人或临时委员会；过渡期可由 创世公民 指定临时决策人 | 防止权力悬空；与签字权承继逻辑一致 |
| **自愿交出** | 创世者书面声明，将决策权移交给联盟内某委员会（如门类理事会联合、公益专项委员会等）；须经 创世公民 过半数确认接收 | 创世者自行判断时机；无人可强制其交出 |
| **创世荣誉监护** | 创世者申请转为创世荣誉监护时，可同步将 1% 决策权移交；见《签字权与法人地位承继规则》§3.3 | 光荣退出的一部分 |

**自愿交出的原因与好处**：创世者认为联盟治理结构已成熟、可放心交由委员会行使时，主动交出决策权。**好处**：① 降低对单一创世者的依赖，文明延续不系于一人；② 增强公信力，对外可宣示「权力已制度化移交」；③ 为创世者光荣退出提供制度出口，与创世荣誉监护呼应；④ 委员会集体决策更易吸纳多方提议、符合公益属性。

**接收方**：须为联盟内经共识成立的委员会（如门类理事会联合、或专门设立的公益决策委员会）；接收时须公告并更新决策权归属记录。

## 五、入账与出账规则

### 5.1 入账

- **1%**：每笔订单 Net Pool 的 1% 直接入池（创世者决策权部分）
- **行业始祖**：待认领池经公投转入后入行业始祖部分
- **捐赠**：按约定入对应子余额
- **记录**：入账标注来源类型，可追溯至 OrderID

### 5.2 出账

| 前提 | 须有对应决策主体的**拨款决议**（含用途、金额、收款方） |
|------|--------------------------------------------------------|
| **校验** | 决议金额 ≤ 对应部分可用余额；否则系统拒绝执行 |
| **执行** | 平台财务按决议划款；划款记录关联 ResolutionID |
| **禁止** | 无决议不划款；越权不执行；超额不划款 |

## 六、披露与审计

| 项目 | 要求 |
|------|------|
| **定期披露** | 每季度公布：公益金池期初/入账/出账/期末；待认领池各品类余额 |
| **审计** | 年度审计；审计方有权查阅完整流水与决议记录 |
| **留存** | 流水与决议至少保留 7 年 |
| **可查** | 全平台参与者可共享查看明细账 |

## 七、与创世分润池的关系

创世分润池（Anchor Pool）= 2% + 2% + 1%：
- **2%** → 创世者
- **2%** → 创世技术官  
- **1%** → **公益积累金**，直接流入**公益金池**（创世者决策权部分）

1% 为创世者与创世技术官各出 0.5% 合成；**提议权与决策权仅归属创世者**，系其制度性影响力来源；创世技术官有知情权与建议权，无决定权；其它节点有提议权与监督权。取之于民，用之于民。
