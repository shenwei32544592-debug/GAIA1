公民参与与贡献记账（Public Participation & Credits）

全称：公共参与范围、贡献分类与兑现原则
用途：定义联盟的可围观范围、贡献分类（C类/G类）、兑现原则、反滥用机制与合规护栏；可嵌入宪章附件或官网制度页
版本：草案 v0.1
性质：协议级条款，与《创世贡献记录与确认机制》《GAIA 社区早期权重计算逻辑》配套
设计意图：自传播 + 自融资 + 自治理的社会机器；参与门槛越低，贡献者总量越大；贡献者越多，制度越像公共事件；越像公共事件，扩散越快

第一条 可围观范围（制度广场）

联盟设立公开可围观的制度广场，任何人可低成本接触联盟规则与运行状态。

类型 内容 访问方式
公共阅读 宪章、理由书、分润逻辑、递归反哺、创意载体规范、账本摘要、治理日志（仅追加、不可删改） 可匿名浏览
公共讨论 论坛、提案区、条款讨论 thread；【艺术家 × 系统需求区】为艺术家→IT 专用直连，见《艺术家与系统需求直连机制》 阅读开放；发言需注册；直连区身份强区隔（仅艺术家可发主帖）
公共旁听 关键治理投票、仲裁结果、里程碑公示 可回看；可匿名观看
身份可见性 签约认可宪章者，论坛内头像/身份可识别；权重参数（Reputation Score、贡献值）公开可查 促进自发维护与建设者涌现；谁在参与、谁有分量人人可见，形成正向示范

原则：让人看得懂、能转述、能加入。围观本身即招募漏斗。身份与权重的可见性，会极大促进自发维护者和建设者的出现。

1.1 身份可见原则

当身份可见、权重可见、贡献可见，秩序会自发产生。

在 GAIA 中，身份不是隐藏的标签，而是公开的承诺。签署宪章者，其身份将被明确标识；长期贡献者，其权重将被持续记录。

我们相信：当身份、声誉与历史行为可被共同见证，公共讨论将自然趋向建设，而无需过度管理或强制干预。

三种心理机制：声誉锁定（实名化身份发言→自觉维护长期声誉）、沉没成本（已签约有权重→不容系统被破坏）、身份自豪（我是创世公民而非路人）。三者叠加，喷子自动消失，建设者浮现。

护城河效应：内部成员本能守护边界；外部攻击被自然免疫；形成「我们」与「非我们」的温和区隔——文明的最小边界。

1.2 权重与发言权的克制设计

原则 内容
权重影响可见度，不影响发言权 所有人都能发言；权重高→发言默认高亮；权重低→发言需被点赞/引用后上浮；得到的是引导，非统治
发言本身不产生权重 提建议≠有贡献；被采纳、被引用、被验证的贡献才产生权重；避免「发言就加分」导致灌水与权力崇拜
不是管理秩序，而是让秩序显形 权重可见=低成本警察；新人自动听高权重者；破坏性言论被群体忽略；无须管理员天天删帖

第二条 贡献分类（C 类 / G 类）

贡献分为两条线，均可低成本参与，兑现方式不同。

2.1 C 类：共识贡献（Consensus Credits）

行为 兑现方式 说明
发言、挑刺、提出可执行建议、参与评议 提案权重、评议权重、声誉等级（影响力） 不直接给钱，只给声誉；为 FCR 评估之参考输入
建议被采纳/被验证 方可进入可兑现池或获得额外权重 采纳/验证后才兑现；未采纳建议仅计参与度

2.2 G 类：增长贡献（Growth Credits）

行为 兑现方式 说明
邀请新人、带来艺术家、带来订单、带来合规渠道 阶段性奖励、未来分润加权、里程碑返还优先级 只奖励真实结果；不做无限层级返利
拉新层级 最多两层（推荐人/推荐人的推荐人），建议仅一层 防止多层分销观感

原则：贡献不是劳务，而是信用资本；个人觉得微不足道，系统层面是复利资产。

第三条 兑现原则（采纳/验证后兑现）

原则 内容
C 类 发言、挑刺计入参与度；仅当建议被采纳或经评议人（Steward）标记为「可执行」，方进入可兑现权重或获得额外声誉
G 类 仅对真实订单、真实签约艺术家、真实合规节点奖励；虚假注册、同 IP 刷量不计入
评议人机制 对建议打标：可执行/不可执行/重复/风险；避免「发言就有分」导致灌水与攻击

原则：把围观转化为秩序，而非噪音场。

第四条 反滥用机制

风险 对策
灌水刷分 单日权重上限；附和权重低于发起；明显无意义帖不计入
虚假拉人 被邀请人须完成签约或实质贡献；同 IP、虚假注册不计入
批评滥用 「有效批评」须经确认人标记；须有具体论点；单纯情绪宣泄不计入
多层分销 拉新奖励最多两层；不设无限层级返利
劣币驱逐良币 仅采纳/验证后兑现；设评议人/Steward 对建议打标

第五条 合规护栏

项目 约定
对外用词 不说空投、代币、挖矿、返佣；说贡献记账、声誉积分、公共信用、里程碑奖励
不构成承诺 贡献记账与声誉积分不构成投资、证券、收益承诺或保证
性质 本机制为参与度与贡献可见性的制度安排；经济权利（如 FCR）须经书面协议单独约定
敏感隔离 与「币圈」「资金盘」「拉人头」等概念保持明确语义与制度隔离

附：可写进理由书的核心表述

GAIA 的增长不是靠广告买来的，而是制度本身带来的： 任何人都可以低成本围观、低成本参与、低成本留下贡献痕迹； 而这些看似微小的发言、挑刺、引荐与见证，会在可追溯的账本里复利增长， 最终把围观者变成公民，把公民变成节点。
