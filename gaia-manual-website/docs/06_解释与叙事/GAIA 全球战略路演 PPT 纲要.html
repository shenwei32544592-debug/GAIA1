每页展开要点（供演讲备注）

P1 · AI 时代的创意主权危机

  • 黑盒算法将创作拆解为数据碎片
  • 单体艺术家失去议价权、失去被看见的概率
  • 联合，是为了将「被动被计算」转变为「主动定规则」

P2 · 从「理想国」到「主权网络」

  • 不是脱离现实的乌托邦
  • 而是在现有世界里拉起一层属于艺术家的独立协议
  • 艺术界的 TCP/IP

P3 · 不证自明

  • 创意是主权行为的延伸
  • 价值的产生者应拥有规则的表决权

P4 · 分配即治理

  • 65% 艺术家 / 5% 创世池 / 20% 运营池 / 10% 风险储备
  • 分配规则写入系统，无人可 override
  • 不是分「利润」，是分「因正义而产生的溢价」

P5 · 共识鉴定

  • 策展人、评论人 = 共识鉴定师
  • 在 DIL 逻辑中注入审美权重
  • 审美定价权的社会资本

P6 · 投资协议的普适性

  • 投资者不拥有互联网，但拥有基于 TCP/IP 的巨型企业
  • GAIA 提供最底层的「艺术信任信用」
  • 投资这个信用基座的全球扩张

P7 · WhoAmI.Art

  • 首个商业实例
  • 签名艺术切入，物理载体 + 数字确权
  • 跑通、赚钱、可复制

P8 · 审计即正义

  • 公益金池、创世池、分配引擎全节点可追溯
  • 审计冗余 = 艺术家的资产安全

P9 · Roadmap

  • 签名 → 绘画/雕塑/手工艺
  • 中国供应链 → 全球交付
  • 制宪会议 → 首届治理 → 宪政自动运行

P10 · Batch 0 招募

  • 创世公民（Batch 0）
  • 制宪会议代表
  • 90 日内贡献者可获 20% 运营池结余的固定比例分配（FCR）
