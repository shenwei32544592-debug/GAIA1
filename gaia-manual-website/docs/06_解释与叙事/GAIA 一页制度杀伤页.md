## 价值如何流动，决定谁终将胜出

| 维度 | 中心化平台 | **GAIA 联盟** |
|------|------------|---------------|
| 价值来源 | 流量聚集 | 原创源头 |
| 平台角色 | 流量地主 / 抽成方 | 基础设施 / 服务节点 |
| 艺术家关系 | 个体商户，相互竞争 | 联盟成员，利益共同体 |
| 竞争机制 | 内卷、自相残杀 | 协作、守望相助 |
| 定价能力 | 被平台与算法决定 | 由联盟议价能力抬升 |
| 抽成逻辑 | 按成交额持续抽血 | 仅收取极低服务费 |
| 规模扩大后 | 商家利润下降 | 成员单位收益上升 |
| 长期结果 | 商家被淘汰、平台垄断 | 非成员被淘汰、联盟扩张 |

---

**一句话**：中心化平台靠「商家互杀」赚钱；GAIA 靠「协作放大价值」生存。

**反馈回路**：平台 = 负反馈（新商家加入→竞争加剧→利润降）；GAIA = 正反馈（新艺术家加入→品牌上升→每人定价能力升）。

**结论**：GAIA 不与平台争流量，而是用更高级的制度**替代平台存在的必要性**。不抽血、不诱导内卷、持续抬升参与者议价能力——胜利靠**人们用脚投票**。
