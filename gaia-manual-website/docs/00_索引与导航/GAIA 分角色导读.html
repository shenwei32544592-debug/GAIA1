GAIA 分角色导读

一句话：GAIA 是 AI 时代内容主权与结算的协议标准——不托管资金、不判案，但能让原创被机器识别、让侵权在商业上不可持续。

使用方式：按你的角色选一条路径，先读「速览」，被吸引后再进「进阶」。

如果你是 CTO / 技术负责人

你最关心的：接入成本、会不会拖垮系统、Resolver 挂了怎么办。

速览（约 2 分钟）：

  • 接入只做两件事：发布时写 GMS 元数据（<1KB）；结算前查一次状态码
  • Resolver 故障 → 自动降级，业务不中断；合规预算 3% 封顶
  • 不托管支付、不改核心链路、不成为单点故障

→ GAIA 1.0 全系统集成规范（Master Spec） §2 工程底座、§6 接入清单

进阶：完整 Master Spec；CTO 技术共识协议

如果你是法务 / 合规负责人

你最关心的：责任边界、是否金融系统、避风港效力、隐私合规。

速览（约 2 分钟）：

  • GAIA 输出「风险信号」，不判定权利；平台是「中立执行方」，非裁决者
  • 不托管资金、不发行代币、非 DAO；等价于「会计旗标 / 合规标识」
  • 报告零内容留存，仅 subject_token 与哈希，满足 GDPR 最小化

→ Master Spec §0 适用声明、§5 法务与审计

进阶：递归反哺与源头确权细则；三项不可谈判权力

如果你是创始人 / 投资人 / 商务

你最关心的：护城河、不可逆点、为什么能成、商业闭环。

速览（约 3 分钟）：

  • 秩序不是被「执行」的，而是被「审计」出来的；不靠道德，靠博弈
  • 违约平台失去 AI 采购、品牌授权、高净值订单；合规即准入
  • 从签名艺术切入（signa.ink）；AI 训练合规是首个不可逆场景

→ GAIA 1.0 创世发布声明；GAIA 一页制度杀伤页

进阶：成立理由书；全球战略蓝皮书；完整思想包

如果你是创作者 / 艺术家

你最关心的：主权、署名、收益回流、我为什么要加入。

速览（约 2 分钟）：

  • 主权始于发布瞬间：写入 GMS，原创轨迹即不可磨灭
  • 65% 归艺术家；递归反哺让二次流转、衍生授权收益回流
  • 争议由同行裁定，非平台独家裁决；证据权重说话

→ GAIA 公共释义版；GAIA 成立理由书（艺术家为何联合）

进阶：创意载体规范；递归反哺细则；FCR 创始贡献者权利

如果你是平台运营 / 产品

你最关心的：风控成本、争议怎么处理、用户教育、接入节奏。

速览（约 2 分钟）：

  • 争议不再靠人工站队；平台执行状态码即可，责任移交给协议
  • 接入分三步：写入 GMS → 监听状态码 → 定义隔离 Vault
  • 合规内容获得高价值订单优先权；灰色内容被市场边缘化

→ 创世发布声明；Master Spec §6 接入清单

进阶：第五编｜平台义务与权力边界；建国路线图

进阶阅读：全貌与细则
