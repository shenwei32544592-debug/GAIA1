# 联盟胚胎发展：成长时间轴与资源映射（战略预参）

**用途**：以胚胎→成人的发育为映射，在成长时间轴上配置各类资源的估算量；展现「投入→成长→扩大→再投入→造血」的完整路径，直至联盟可自我运转  
**性质**：战略预参；所有数值为估算，供决策参考  
**依据**：《有效路径与时间节点安排》《建国方案：小步快走与多期 FCR 设计（草案）》《IT 基础设施甘特图与成本估算》《基建期内生融资：0到1现金流预测模型》

**计算标准**（中国头批成功经典）：

- **10 位艺术家**，每人每天 **50 单**
- **平均单订单可分配利润**：100 元（Net Pool，扣除硬成本后）
- **平台日单量**：500 单；**平台日 Net Pool**：5 万元；**平台月 Net Pool**：150 万元

### 0.4 简化造血路径与互信驱动收益再借入

**核心判断**：造血不必等待完整 IT。少量关键节点接入 + 人工记账，即可打通闭环；前期 IT 功能可精简。

| 要素 | 设计 |
|------|------|
| **节点门槛** | 另加不多几个节点即可打通造血；不依赖大规模 IT 部署 |
| **记账方式** | IT 未完备时，**手动记账**可接受；合作机制优先于技术基建 |
| **互信锁定** | 合作机制中**互信一旦锁定**，艺术家、供应商、协作伙伴可能愿将所赚**大部分借给联盟**用于快速发展 |
| **成功信号** | **明确的成功信号**（首单闭环、首笔跨境、FCR 流水、订单爬坡等）是触发再借入意愿的关键 |

**与既有机制关系**：

- **建国者共建金**：建国者小额投入 + 透明示范；可扩展为「成功信号后收益再借入」的入口
- **基建期艺术家共投**：艺术家自愿将分润借给联盟；互信 + 成功信号下，借出比例可显著高于基准
- **借出 vs 分润**：借给联盟须有**还款条件、顺序、期限**；与 FCR 分润权分离，不产生永久统治权

**路径示意**：互信锁定 → 成功信号可验证 → 艺术家/供应商/伙伴收益再借入 → 造血临界提前达成。

**重要节点优先认购**：发展基建基金**优先接受**艺术家、供应商、类帐户的投资认购；其与系统成功高度相关，有利于合作机制发展更平稳、更快速。*详见《建国方案》§3.5、《基建期艺术家共投机制》*

**结构性稳定**：机制内配合方皆知无人可跳单独干；利益流咬合，天然生死同盟，根本无需担心互踩背叛。此乃**结构性稳定**，非依赖规则惩戒的制度性稳定。*详见《建国方案》§3.6*

### 0.5 小步快走与多期 FCR（可选路径）

**建国锁定期可延后**，以造血循环周期为锚点，小步快走；多期配置不同 FCR 结构。  
*详见《建国方案：小步快走与多期 FCR 设计（草案）》。*

| 期 | 触发条件 | FCR 结构 | 建国锁定 |
|----|----------|----------|----------|
| 期 0 胚胎 | 协议 + 001 | 无 / 意向 | 否 |
| 期 1 首单 | 能接单、人工记账 | FCR-1 简化 | 否 |
| 期 2 爬坡 | 50–150 单/天 | FCR-2 扩大 | 否 |
| 期 3 造血临界 | ~500 单/天 | FCR-3 完整；Batch 0 锁定 | **是** |
| 期 4 成人 | 造血稳定 | 分配表冻结 | 已锁定 |

**建国锁定实质触发**：造血临界达成 + 有权重节点投票通过；Day 90 为参考，非刚性。

### 少年期（18–36 月）

**定义**：分配引擎连续运行；门类理事会、裁定者就位；治理层可独立运转。

| 资源类型 | 投入量 | 成本(￥) | 产出 |
|----------|--------|----------|------|
| **IT** | GAIA 完整核心（Consensus、Physical、Alliance） | 40万–70万 | 治理、联盟接口 |
| **治理** | 门类理事会、裁定者运营与激励 | 5万–15万 | 治理层就位 |
| **运营** | 扩展、多区域、品牌 | 20万–50万 | 规模扩大 |
| **收入** | GMV 稳定增长 | **流入** | 覆盖大部分 |
| **小计** | — | **65万–135万**（净投入递减） | |

**累计**：223万–455万。  
**关键指标**：分配引擎 24 月无人工干预、门类理事会≥3 门类、裁定者≥3。  
**接近造血**：收入逐步覆盖运营；扩展投入仍依赖外部或结余再投。

---

### 成人期（36 月+）

**定义**：收入 ≥ 运营成本 + 合理扩展投入；制度自动运转；可持续扩张。

| 资源类型 | 状态 | 说明 |
|----------|------|------|
| **运营** | 收入覆盖 | 日常运营由 GMV 分润与平台抽成支撑 |
| **扩展** | 内生或可控 | 基建池、建国回补、艺术家共投可支撑部分扩张 |
| **治理** | 自动 | 裁定者、门类理事会按规则运转 |
| **再投入** | 可选 | 新市场、新品类、联盟接口；由结余或融资决定 |

**造血判定**：  
- 月均运营成本 ≤ 月均平台收入（20% 运营池 + 5% 创世池等）  
- 无需外部输血即可维持运转与适度扩张

## 三、投入–成长–扩大–再投入循环

```
                    ┌──────────────────────────────────┐
                    │           再投入（扩张）             │
                    │  IT扩展 │ 获客 │ 多区域 │ 联盟接口  │
                    └────────────┬───────────────────────┘
                                 │
                                 ▼
    ┌─────────────┐         ┌─────────────┐         ┌─────────────┐
    │    投入     │ ──────► │    成长     │ ──────► │    扩大     │
    │  IT│物理│   │         │  产出可见    │         │  规模↑      │
    │  合规│获客  │         │  指标达成    │         │  节点↑      │
    └─────────────┘         └──────┬──────┘         └──────┬──────┘
          ▲                        │                        │
          │                        │  收入回流              │
          │                        ▼                        │
          │                 ┌─────────────┐                  │
          └────────────────│   造血     │◄─────────────────┘
                            │ 收入≥成本  │
                            └─────────────┘
```

**逻辑**：每次投入带来成长→成长带来扩大→扩大产生收入→收入支撑再投入；当收入足以覆盖成本，即造血。

## 七、关键决策点

| 时点 | 决策 | 依据 |
|------|------|------|
| **Day 90** | 建"国"是否成功；FCR 窗口是否关闭 | 投票结果 |
| **学步期末** | 是否追加投入推儿童期 | 首单闭环、商业验证 |
| **儿童期末** | 是否推少年期、GAIA 完整核心 | FCR 流水、艺术家留存 |
| **少年期末** | 是否宣布造血、风险解除 | 分配引擎连续、治理就位 |
| **成人期** | 扩张节奏、联盟接口优先级 | 结余、战略 |

## 八、弹性与风险

| 情形 | 应对 |
|------|------|
| **收入低于预期** | 压缩扩展、延后少年期、优先运营成本覆盖 |
| **投入超支** | 分阶段砍非核心；IT 可采购 Edge、延后 GAIA 完整 |
| **建"国"未成功** | 延 Day 90、调整门槛、再私邀 |
| **造血提前** | 订单超预期时，可提前进入再投入循环；**互信锁定 + 成功信号**下，艺术家/供应商/伙伴收益再借入可显著缩短达标周期 |
